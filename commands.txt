📚 Vivlio Starter - Build System

主なコマンド:
  vs build                       - 全ファイルをビルド
  vs build <chapter_name>        - 指定した章のみをビルド
  vs open                        - 生成されたPDFを開く(macOS専用)
  vs pdf:compress                - 生成されたPDFを圧縮
  vs doctor                      - 必要ツールの診断とセットアップ(macOS)
    （Xcode Command Line Tools / Homebrew / Node.js / qpdf / pdfinfo / gs / ImageMagick 等）
    --fix                        # 不足ツールを自動インストール(Homebrew)。Homebrew が無ければ自動導入を案内
                                 # macOS では CLT 未導入時にインストーラを起動し、完了を待機
    -y, --yes                    # 確認を省略（Homebrew 導入や npm -g 実行時の確認をスキップ）

# ビルドオプション:
--high                # 高画質でビルド（画像の品質優先）
--medium              # 標準画質でビルド（デフォルト）
--low                 # 低画質でビルド（ファイルサイズ優先）
--no-resize           # 画像のリサイズ/最適化をスキップ
--no-compress         # PDFの圧縮をスキップ
--no-clean            # 中間ファイルのクリーンアップをスキップ
--log[=level]         # ログレベルを指定（error/warn/info/debug、既定は info）

章の管理:
  vs create <chapter_name>       - 新しい章を作成
  vs delete <chapter_name>       - 指定した章を削除
  vs rename <old> <new>          - 章名・付録名/番号を変更
                                （例: 11-install → 12-setup, 21 → 32）
  vs renumber                    - 章番号・付録番号を整列
  vs clean                       - ビルド生成物をクリーンアップ

プロジェクト作成:
  vs new <name>                  - 新規書籍プロジェクトを作成
    (既定) 依存ツールの自動インストールを実行し、確認を省略します
    --interactive                # 対話的に確認しながら実行
    --manual-install             # doctor の自動実行を無効化

ヘルプ:
  vs help                        - このヘルプを表示
  vs --version                   - バージョン情報を表示

vs [-h/--help] [--version] [-v/--verbose] <command>
	Vivlio Starter CLI (Samovar)

	[-h/--help]     Vivlio Starter のヘルプを表示                                                                                                                                                                                                                                                                            
	[--version]     バージョン情報を表示                                                                                                                                                                                                                                                                                        
	[-v/--verbose]  冗長出力を有効化                                                                                                                                                                                                                                                                                          
	<command>       One of: help, new, build, clean, delete, doctor, entries, create, create:titlepage, create:colophon, create:legalpage, rename, renumber, pre_process, convert, post_process, toc, pdf, pdf:compress, resize, resize:high, resize:medium, resize:low, index, index:auto, index:apply, open, import.  (default: help)

	help [-h/--help]
		Vivlio Starter の主要コマンド一覧を表示します

		[-h/--help]  ヘルプを表示

	new <name> [--[no]-auto-install] [--interactive] [--manual-install] [-h/--help]
		新しい書籍プロジェクトを作成します

		<name>                 プロジェクト名（必須）                      (required)     
		[--[no]-auto-install]  必要ツールを自動インストール (macOS Homebrew)  (default: true)
		[--interactive]        対話的に確認しながら実行                   
		[--manual-install]     doctor の自動実行を無効化               
		[-h/--help]            このコマンドの使い方を表示                  

	build <targets...> [--[no]-resize] [--high] [--medium] [--low] [--[no]-compress] [--[no]-clean] [-n/--dry-run] [--log <level>] [--force] [--no-cache] [-h/--help]
		書籍全体または指定章をビルドします

		<targets...>       ビルド対象（章番号 / 範囲 / ベース名）           (default: [])  
		[--[no]-resize]    画像最適化を行う（--no-resize で無効）        (default: true)
		[--high]           画像最適化プリセット: 高品質                
		[--medium]         画像最適化プリセット: 中品質                
		[--low]            画像最適化プリセット: 低品質                
		[--[no]-compress]  PDF圧縮を行う（--no-compress でスキップ）  
		[--[no]-clean]     中間生成物をクリーンアップ（--no-clean でスキップ）  (default: true)
		[-n/--dry-run]     実行せずにビルド予定のみを表示                
		[--log <level>]    ログレベルを指定（error/warn/info/debug）
		[--force]          タイトル/リーガル/奥付を強制再生成             
		[--no-cache]       キャッシュを無効化（--force と同義）         
		[-h/--help]        このコマンドの使い方を表示                  

	clean [--purge/-P] [--cache/-C] [--cover] [--generated-images] [-h/--help]
		生成物やキャッシュを削除します

		[--purge/-P]          生成物（PDF含む）をすべて削除します  
		[--cache/-C]          キャッシュのみを削除します        
		[--cover]             生成されたカバー画像のみを削除します   
		[--generated-images]  生成された扉絵/装飾などの画像を削除します
		[-h/--help]           このコマンドの使い方を表示        

	delete <targets...> [--dry-run/-n] [--force/-f] [--yes/-y] [--verbose/-v] [-h/--help]
		指定した章の Markdown と画像を削除します

		<targets...>    削除対象（章番号/レンジ/ファイル名）  (default: [])
		[--dry-run/-n]  変更せずに削除予定のみを表示     
		[--force/-f]    確認なしで削除を実行         
		[--yes/-y]      [互換] --force と同じ意味 
		[--verbose/-v]  冗長ログを表示            
		[-h/--help]     このコマンドの使い方を表示      

	doctor [--fix] [--yes/-y] [-h/--help]
		環境診断と不足ツールの自動セットアップを行います

		[--fix]      不足ツールを自動インストール (macOS Homebrew)
		[--yes/-y]   確認プロンプトをスキップ                   
		[-h/--help]  このコマンドの使い方を表示                  

	entries <tokens...> [-h/--help]
		HTML から entries.js (ES Module) を生成します

		<tokens...>  対象 HTML のスラッグまたはファイルパス（省略時は *.html 全体）  (default: [])
		[-h/--help]  このコマンドの使い方を表示                         

	create <names...> [-h/--help]
		章ファイルと画像ディレクトリを生成します

		<names...>   作成する章スラッグ（複数可）  (default: [])
		[-h/--help]  このコマンドの使い方を表示 

	create:titlepage [--force/-f] [-h/--help]
		config/book.yml からタイトルページを生成します

		[--force/-f]  既存ファイルを強制上書き 
		[-h/--help]   このコマンドの使い方を表示

	create:colophon [--force/-f] [-h/--help]
		config/book.yml から奥付 (_colophon.md) を生成します

		[--force/-f]  既存ファイルを強制上書き 
		[-h/--help]   このコマンドの使い方を表示

	create:legalpage [--force/-f] [-h/--help]
		config/book.yml の legal 設定からリーガルページを生成します

		[--force/-f]  既存ファイルを強制上書き 
		[-h/--help]   このコマンドの使い方を表示

	rename <arguments...> [-n/--dry-run] [--force/-f/-y] [--chapter-step/-S <step>] [--step <step>] [-h/--help]
		章のスラッグ/番号を変更します

		<arguments...>              OLD と NEW を指定。省略すると一括連番モード        (default: [])
		[-n/--dry-run]              変更予定のみ表示（実行しない）                 
		[--force/-f/-y]             確認なしで変更を実行                      
		[--chapter-step/-S <step>]  章番号の刻み幅を指定（既定: 1）               
		[--step <step>]             [互換] 章番号の刻み幅（--chapter-step と同義）
		[-h/--help]                 このコマンドの使い方を表示                   

	renumber <arguments...> [-n/--dry-run] [--force/-f/-y] [--chapter-step/-S <step>] [--step <step>] [-h/--help]
		章番号を一括で付け直します（rename の別名）

		<arguments...>              OLD と NEW を指定。省略すると一括連番モード        (default: [])
		[-n/--dry-run]              変更予定のみ表示（実行しない）                 
		[--force/-f/-y]             確認なしで変更を実行                      
		[--chapter-step/-S <step>]  章番号の刻み幅を指定（既定: 1）               
		[--step <step>]             [互換] 章番号の刻み幅（--chapter-step と同義）
		[-h/--help]                 このコマンドの使い方を表示                   

	pre_process <tokens...>
		Markdownファイルの前処理を行います

		<tokens...>  対象ファイル（省略時は全ファイル）

	convert <tokens...>
		MarkdownをHTMLに変換します

		<tokens...>  対象ファイル（省略時は全ファイル）

	post_process <tokens...>
		HTMLファイルのポスト置換処理を行います

		<tokens...>  対象ファイル（省略時は全ファイル）

	toc <htmls...>
		目次HTMLを生成します

		<htmls...>  対象HTMLファイル（省略時は自動検出）

	pdf <output> [-h/--help]
		PDFを生成します

		<output>     出力ファイル名（省略時は設定に従う）
		[-h/--help]  このコマンドの使い方を表示     

	pdf:compress <input> <output> [-h/--help]
		生成済みPDFを圧縮します

		<input>      入力PDFファイル    
		<output>     出力PDFファイル    
		[-h/--help]  このコマンドの使い方を表示

	resize [-f/--force] [--high] [--low] <dir>
		画像をWebPに変換します（標準品質）

		[-f/--force]  既存ファイルも強制再生成
		[--high]      高品質プリセットを使用 
		[--low]       軽量品質プリセットを使用
		<dir>         対象ディレクトリ      (default: "images")

	resize:high <dir>
		画像を高品質WebPに変換します

		<dir>  対象ディレクトリ  (default: "images")

	resize:medium <dir>
		画像を標準品質WebPに変換します

		<dir>  対象ディレクトリ  (default: "images")

	resize:low <dir>
		画像を軽量品質WebPに変換します

		<dir>  対象ディレクトリ  (default: "images")

	index [-h/--help]
		索引機能のサブコマンドを表示します

		[-h/--help]  このコマンドの使い方を表示

	index:auto [-v/--verbose] [-h/--help] <files...>
		候補抽出・分類・_index_review.md 生成

		[-v/--verbose]  詳細出力          
		[-h/--help]     このコマンドの使い方を表示 
		<files...>      対象ファイル（省略時は全章）

	index:apply [-v/--verbose] [-h/--help]
		レビュー結果を index_terms.yml に適用

		[-v/--verbose]  詳細出力         
		[-h/--help]     このコマンドの使い方を表示

	open [-v/--verbose] [-h/--help]
		生成されたPDFを開く（macOS専用）

		[-v/--verbose]  冗長出力         
		[-h/--help]     このコマンドの使い方を表示

	import <arguments...> [--force] [-h/--help]
		Re:VIEW Starter プロジェクトをインポートします

		<arguments...>  Re:VIEW Starter プロジェクトのディレクトリ  (default: [])
		[--force]       確認プロンプトをスキップして実行             
		[-h/--help]     このコマンドの使い方を表示
