## Vivlio Starter とは？

Vivlio Starter は、Markdown から高品質な電子書籍（PDF）を生成するための CLI ツールです。以下の特徴を持っています：

- **直感的な CLI コマンド**: `vs build` の一コマンドでビルド完了
- **一元的な設定管理**: `config/book.yml` ですべての設定を管理
- **スマートなスタイル適用**: ファイルタイプに応じた自動スタイル切り替え
- **自動化されたワークフロー**: 前処理から PDF 生成までを完全自動化

### インストールとセットアップ

まずは Vivlio Starter のインストールから始めましょう。必要なツールのインストールから、最初のプロジェクト作成までを詳しく説明します。

```bash
# Ruby のインストール（macOS）
bin/install-ruby.zsh

# Vivlio Starter のインストール
gem install vivlio-starter

# 新しいプロジェクトの作成
vs new my-book
cd my-book
```

### 執筆の基本

Markdown で原稿を書く方法から、Vivlio Starter 独自の便利な機能までを学びます。

```markdown
# はじめての章

これは章の見出しです。

## 節の見出し

節の内容をここに書きます。

![](images/sample.png){width=50%}

画像も簡単に挿入できます。
```

### データ駆動なコンテンツ

Vivlio Starter の強力な機能である QueryStream を使って、データからコンテンツを自動生成する方法を学びます。

```markdown
= books

# 書籍一覧

`_book.md` テンプレートを使って、すべての書籍情報を自動的に表示します。
```

### 美しいデザインとカスタマイズ

CSS 変数によるテーマ設定や、カスタムスタイルの追加方法を学びます。

```yaml
# config/book.yml
theme:
  color: blue
  style: simple
```
