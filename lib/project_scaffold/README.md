# テンプレートファイル

このディレクトリには、新しい章を作成する際に使用されるテンプレートファイルが含まれています。

## ファイル一覧

### `chapter_template.md`
新しい章を作成する際のテンプレートファイルです。

#### プレースホルダー
- `{{TITLE}}`: 章のタイトルに置換されます

#### カスタマイズ方法
1. `chapter_template.md`を編集して、お好みのテンプレートに変更
2. プレースホルダー `{{TITLE}}` は必ず残してください
3. 変更後、`vs create <filename>` で新しいテンプレートが使用されます

#### 使用例
```bash
vs create 21-quantum-physics
```

上記コマンドで、`{{TITLE}}` が "Quantum physics"（量子物理学）に置換されたファイルが生成されます。

## 注意事項
- テンプレートファイルが見つからない場合は、デフォルトのシンプルなテンプレートが使用されます
- 文字エンコーディングはUTF-8で保存してください
