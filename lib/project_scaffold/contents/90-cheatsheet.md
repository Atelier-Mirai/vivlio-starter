# 記法早見表

:::{.chapter-lead}
本編で紹介した記法を「やりたいこと」から逆引きできる早見表です。書き方の細かい約束事や実行例は、「解説章」列の章を参照してください。「執筆チュートリアル」は「Markdown 執筆チュートリアル」、「拡張リファレンス」は「拡張記法リファレンス」の章の略記です。
:::

### 文章の基本

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| 見出しを付ける | `# 見出し`（`#` の数 1〜6 でレベル） | 執筆チュートリアル |
| 太字にする | `**太字**` | 執筆チュートリアル |
| 斜体にする | `*斜体*` | 執筆チュートリアル |
| 取り消し線を引く | `~~取り消し線~~` | 執筆チュートリアル |
| ルビ（振り仮名）を振る | `{相対性理論\|そうたいせいりろん}` | 執筆チュートリアル |
| キーボードのキーを示す | `〘Ctrl〙 + 〘S〙` | 拡張リファレンス |
| 改行する | 行末で改行するだけ（ハード改行）。`<br>` でも可 | 執筆チュートリアル |
| 段落を分ける | 空行で区切る | 執筆チュートリアル |
| 記号をそのまま表示する | `\*` のように `\` を前置してエスケープ | 執筆チュートリアル |
| 改ページする | `---` だけの行を置く | 執筆チュートリアル |
:::

### リスト

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| 箇条書き | `- 項目`（字下げで入れ子。マーカーは ● ○ ・ と自動変化） | 執筆チュートリアル |
| 番号付きリスト | `1. 項目` | 執筆チュートリアル |
| 英字・ローマ数字の番号 | `a.` `(A)` `i)` などのマーカーで書き始める（fancy list） | 執筆チュートリアル |
| 複合番号（1.1 / 1.2） | 番号リストを `:::{.outline-list}` で囲む | 拡張リファレンス |
| 用語と説明を並べる | 用語の次の行に `: 説明`（定義リスト） | 執筆チュートリアル |
| ▶ 付きの補足リスト（青コメ） | `- ▶ 補足` | 拡張リファレンス |
| ❶ 付きの手順リスト（赤コメ） | `- ❶ 手順` | 拡張リファレンス |
:::

### コードとターミナル

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| インラインコード | バッククォート 1 つで囲む | 執筆チュートリアル |
| コードブロック | バッククォート 3 つのフェンスで囲み、言語名を指定（例: `ruby`） | 執筆チュートリアル |
| ファイル名を表示する | 言語名に続けて `ruby:app.rb` | 執筆チュートリアル |
| 行番号を途中から始める | `ruby:foo.rb#L5` | 拡張リファレンス |
| 外部ファイルを取り込む | フェンスに `include:sample.rb`（`:10-20` で範囲指定） | 拡張リファレンス |
| コメント行を強調する | コメント内に `[!]` を書く（例: `# [!] 注目`） | 拡張リファレンス |
| ターミナル入力を示す | `:::{.terminal}` で囲み、`$ コマンド` を書く | 拡張リファレンス |
| 実行結果・出力例を示す | `:::{.output}` で囲む | 拡張リファレンス |
:::

### 表

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| 表を作る | 縦棒で列を区切り、`---` の行でヘッダーと本体を分ける | 執筆チュートリアル |
| セルの配置を変える | 区切り行を `:--`（左）`:-:`（中央）`--:`（右）にする | 執筆チュートリアル |
| セルを横に結合する | 結合したいセルを空にして縦棒を続ける（`\|\|`） | 拡張リファレンス |
| ヘッダーを複数行にする | 区切り行の上に複数の行を書く | 拡張リファレンス |
| セル内に縦棒を書く | 縦棒の前にバックスラッシュを置いてエスケープする | 拡張リファレンス |
| 行の多い長い表 | `:::{.long-table}` で囲む（文字が小さめに詰まる） | 拡張リファレンス |
| 横に広い表を回転する | `:::{.rotate-table}`（`scale` `shift-y` で微調整） | 拡張リファレンス |
:::

### 画像・図

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| 画像を貼る | `![代替テキスト](画像.png)` | 執筆チュートリアル |
| サイズ・配置を変える | `![](画像.png){width=50% align=center}` | 拡張リファレンス |
| キャプションと図番号を付ける | 画像の直前の行に `** タイトル @ラベル **` | 拡張リファレンス |
| 画像の周囲を切り落とす | `![](画像.png){crop="30 100"}` | 拡張リファレンス |
| 写真をグリッドで並べる | `:::{.pictures}` の中に画像を並べる | 拡張リファレンス |
| 画像を 2 枚横に並べる | `:::{.image-group}` | 拡張リファレンス |
| 画像と本文を横に並べる | `:::{.img-text}`（`.img-text2` `.text2-img` など比率違いあり） | 拡張リファレンス |
| 本文を画像に回り込ませる | `:::{.sideimage-right}` / `:::{.sideimage-left}` | 拡張リファレンス |
| スクリーンショットに注釈を描く | `:::{.showcase}` の中に `rect:1 …` / `pointer:2 …` を書く | 拡張リファレンス |
| 罫線図・アスキーアートを載せる | `:::{.diagram}` で囲む（行番号なしの囲み枠） | 拡張リファレンス |
:::

### リンク・脚注・数式

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| リンクを張る | `[テキスト](URL)` | 執筆チュートリアル |
| URL をそのままリンクにする | `<https://example.com>` | 執筆チュートリアル |
| 脚注（参照型） | 本文に `[^id]`、文末に `[^id]: 内容` | 執筆チュートリアル |
| 脚注（インライン） | `^[短い補足]` | 執筆チュートリアル |
| 数式を書く | インラインは `$a^2 + b^2 = c^2$`、別行立ては `$$…$$` | 執筆チュートリアル |
:::

### レイアウト・余白

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| ブロックごと寄せる | `:::{.align-left}` / `.align-center` / `.align-right` | 拡張リファレンス |
| 文字の揃えだけ変える | 段落末尾に `{.text-right}` など（`.text-left` / `.text-center` も） | 拡張リファレンス |
| 二段組にする | `:::{.text-2dan}` | 拡張リファレンス |
| 段落の下に余白を空ける | 段落末尾に `{.aki}`（1 行分）/ `{.aki2}`（2 行分） | 拡張リファレンス |
| 余白を微調整する | `@vspace:1.5lh`（負の値で詰める。単位は `lh` `mm` など） | 拡張リファレンス |
| クラスを組み合わせる | `:::{.column .align-center}` のように併記 | 拡張リファレンス |
| 独自の装飾を作る | `:::{.myclass}` ＋ `stylesheets/custom.css` に定義 | 拡張リファレンス |
:::

### 参照・索引・データ

:::{.long-table}
| やりたいこと | 書き方 | 解説章 |
|:---|:---|:---|
| 図・表・コードを参照する | 定義側 `** タイトル @ラベル **`、参照側 `@ラベル` | クロスリファレンス |
| 索引・用語集に登録する | `[用語\|読み]`（読みは省略可） | 索引・用語集機能 |
| 索引候補を自動抽出する | `vs index:auto` → レビュー → `vs index:apply` | 索引・用語集機能 |
| データを展開する | `= books \| tags=ruby \| 3`（`data/` の YAML＋テンプレート） | データ展開機能 |
:::

:::{.memo}
この早見表は「そういえばあの記法、どう書くのだったか」を思い出すためのものです。fancy list の空白規則や表の結合の細則など、記法ごとの約束事は解説章のほうにまとめてあります。
:::
