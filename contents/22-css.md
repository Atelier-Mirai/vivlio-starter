## CSS(Cascading Style Sheets)とは何か

これから、CSSを使って、じゃんけんゲームの見栄えを整えていきます。CSSとは何か、把握しておきましょう。

開発者のためのウェブ技術情報が集積されているサイトMDNに、[CSSの基本](https://developer.mozilla.org/ja/docs/Learn/Getting_started_with_the_web/CSS_basics) という優れた記事があり、そこには次のように書かれています。

![](css-declaration.webp)

CSS (Cascading Style Sheets) は、ウェブページのスタイルを設定するコードです。
CSSは、HTMLの要素を選択的にスタイル付け(装飾)するために使用します。
例えば、右の CSS は段落のテキストを選択し、色を赤に設定します。


### CSS 規則セットの構造

赤い段落テキストの CSS コードを分解して、その仕組みを理解してみましょう。

全体の構造は規則セットと呼びます (単に規則とも呼ばれます)。

- **セレクタ (Selector)**
    これは規則セットの先頭にある`HTML`要素名です。これはスタイルを設定する要素 (この例の場合は `<p>`要素) を定義します。別の要素をスタイル付け(装飾)するには、セレクタを変更してください。

- **宣言 (Declaration)**
    `color: red;` のような単一の規則です。これは要素のプロパティのうち、スタイル付けしたいものを指定します。

- **プロパティ (Property)**
    これらは、`HTML`要素を(どのように)スタイル付け(装飾)するかを指定します (この例では、`color`は `<p>`要素のプロパティです)。`CSS`では、規則の中で影響を与えたいプロパティを選択します。

- **プロパティ値 (Property value)**
    プロパティの右側には `:(コロン)` の後に「プロパティ値」を記述します。各プロパティにはそれに応じた様々なプロパティ値を設定することができます。例えば、`color`の値は`red`以外にも沢山の値を設定できます。



[tip] <b>まとめ</a>

* セレクタを除き、それぞれの規則セットは中括弧 (`{}`) で囲みます。
* それぞれの宣言内では、コロン (`:`) を使用してプロパティと値を分離します。
* それぞれの規則セット内では、セミコロン (`;`) を使用して、それぞれの宣言と次の規則を区切ります。

[/tip]


## CSSでの色指定

![](rgb.webp)

先ほどの例では、色を指定する際に `color: red;` と、色の名前を書きました。人の目は数百万色を区別できると言われています。これら全てに色名を付けることはできませんので、CSSでは、光の三原色 赤(Red)・緑(Green)・青(Blue)の組み合わせで色を表現する「RGBカラー」が良く用いられます。[^65]

光を重ねるほど明るくなり、三色が重なる部分は白になります(加法混色)。
色の三原色と呼ばれる 黄・赤紫・青緑は、重ねるほどに暗くなります(減法混色)。
光の三原色「赤・緑・青」と、色の三原色「黄・赤紫・青緑」、相補性を成しています。

[^65]: [基本的な色指定であるRGB値](https://www.webcolordesign.net/color_basic/hsb_basic/rgb_color.html)より、引用・改変。

赤(Red)・緑(Green)・青(Blue)のそれぞれを、「点灯している」「消灯している」の二段階で組み合わせると $2^3=8$色を表現可能できます。中間色を表現するために 各色に256段階の輝度を持たせることとすると $256^3 = 16,777,216$ 色を表現可能です。

色の指定は、`#(シャープ)`に続けて、この256段階を16進数に変換した表記が良く用いられます。例えば赤は`#ff0000`、緑は`#00ff00`、青は`#0000ff`です。赤と緑を合わせると黄`#ffff00`になります。

`color: red;`に代えて`color: #ff0000;`と書くことができます。

```css:janken.css

=====================================================================*/

/* CSSカスタムプロパティを定義して、和の色を使えるようにする
---------------------------------------------------------------------*/
:root {
  --kyohiiro:          #ff251e; /* 京緋色(きょうひいろ) */
  --shinonomeiro:      #f19072; /* 東雲色(しののめいろ) */
  --nanohanairo:       #ffec47; /* 菜の花色(なのはないろ) */
  --sanaeiro:          #67a70c; /* 早苗色(さなえいろ) */
  --amairo:            #2ca9e1; /* 天色(あまいろ) */
  --utsushiiro:        #3d6eda; /* 移色(うつしいろ) */
  --botaniro:          #e7609e; /* 牡丹色(ぼたんいろ) */
  --ayameiro:          #674196; /* 菖蒲色(あやめいろ) */
  --sakurairo:         #fef4f4; /* 桜色(さくらいろ) */
  --momijiiro:         #a61017; /* 紅葉色(もみじいろ) */
  --nibiiro:           #9ea1a3; /* 鈍色(にびいろ) */
  --kurohairo:         #0d0d0d; /* 黒羽色(くろはいろ) */

  --harukazeiro:       transparent; /* 春風色(はるかぜいろ) */
}

/* 基本設定(リセットCSS)
---------------------------------------------------------------------*/
* {                       /* 全ての要素(*)を対象に装飾方法を指定する */
  margin: 0;              /* 余白を0にする */
  box-sizing: border-box; /* 要素の幅を制御しやすくする */
}
```


### コメント（説明書き）

`/*` と `*/` で囲まれた部分は「コメント」です。
コメントは、ブラウザが CSS を解釈する際には読み飛ばされ、表示には影響を与えませんので、
「見出し」や「説明書き」として、使うことができます。

CSS は、HTML のそれぞれの要素について、文字の大きさや色指定や配置など、事細かに装飾していくため、長くなります。
表題を入れて、視覚的にも、大まかに何について書かれている部分なのか判別しやすくとともに、それぞれのコードの意味が分かりやすいよう、一行一行、説明を付けています。

### ユーザーエージェントスタイルシート と リセットCSS

ブラウザには「ユーザーエージェントスタイルシート」というスタイルシートが標準で適用されています。そのため全くCSSを記述せずとも、余白の調整などある程度の装飾が行われ読みやすくなっています。そして、ブラウザがスタイルシートを適用してくれているのですが、自分でデザインを創り上げる際には少しおせっかいに感じることもあります。

そこで、不要・不都合なスタイルを上書きし、初期化したいという欲求が生じます。そのためのCSSを「リセットCSS」と言い、先の基本設定はその一例です。

自身で作成するのも一法ですが、多くの先人たちの手により、リセットCSSが作成されていますので、それを使うのも良いでしょう。 [^68]

[^68]: [mayank99/reset.css](https://github.com/mayank99/reset.css/blob/main/package/index.css)など、多くのリセットCSSがあります。


### CSSグリッドレイアウト

CSSグリッドレイアウトとは、ウェブページのレイアウトを簡単に構築できるCSSの機能です。


[tip] <b>特徴</a>

* 行と列からなるグリッド（格子）状の枠組みを定義できる。
* グリッド内の要素の配置や大きさを柔軟に制御できる。
* 複雑なレイアウトを簡単に実現できる。
* レスポンシブデザインにも対応し、デバイスに応じた最適なレイアウトが実現できる。

[/tip]


現在は、特にCSSを記述していないので、HTMLで書かれた順番に、上から下、左から右と表示されます。CSSグリッドレイアウトを使うことで、見出しや画像などそれぞれの要素を好きな場所に配置することができるようになります。

次のように、CSSを書きましょう。

```css:janken.css
/* CSSグリッドレイアウトで ページ全体の配置設定する
---------------------------------------------------------------------*/
body {            /* 全ての要素の親であるbody要素を対象に */
  display: grid;  /* グリッド(格子)を使ったレイアウトを行う */

  /* 一行目はtitleなど 五行六列の各領域に名前を付ける
     また、行の高さ、列の幅も同時に設定する */
  grid-template:
    "title    title    title    title    title    title   " auto
    ".        computer computer computer computer .       " auto
    "note     note     note     note     note     note    " auto
    "guu      guu      choki    choki    paa      paa     " auto
    "score    score    score    play     play     play    " auto
    / 1fr     1fr      1fr      1fr      1fr      1fr;

  gap: 10px;               /* 行と列の間にgap(隙間)を10px入れる */

  inline-size: 100vw;      /* 幅を画面の幅にする */
  max-inline-size: 430px;  /* 最大幅を iPhone 15 Pro Max の横幅にする */
  block-size: 100svh;      /* 高さを画面の高さにする */
  max-block-size: 932px;   /* 最大高を iPhone 15 Pro Max の横幅にする */
  margin-inline: auto;     /* iPad で見る人の為に、左右に余白を設ける */

  background: var(--sakurairo); /* 背景色は桜色にする */
  color: var(--kurohairo);      /* 文字色は黒羽色にする */

  > * {                    /* body の すべての子要素を対象に */
    place-self: center;    /* 升目の中央に配置するよう指定する */
  }
}
```

34行目の `display: grid;` で、グリッド(格子)を使ったレイアウトを行うことを宣言しています。

38行目〜44行目に書かれている `grid-template:`から始まる記述により、`<body>`要素全体のレイアウトを定義しています。

```
"title    title    title    title    title    title   " auto
".        computer computer computer computer .       " auto
"note     note     note     note     note     note    " auto
"guu      guu      choki    choki    paa      paa     " auto
"score    score    score    play     play     play    " auto
```

と記述しているので、縦5行・横6列 の枠を用意しています。また、右端に記述されている `auto` は、各行の高さを自動で設定する為のものです。

44行目に書かれている次の記述についてですが、

```
    / 1fr     1fr      1fr      1fr      1fr      1fr;
```

`/` に続けて、`1fr` と6回書いています。これにより、6列全てを同じ幅(`1fr`)に設定できます。


それでは、どのように見えるか、確認してみましょう。[^69]
開発者ツールの右下「レイアウト」の下に「グリッドをオーバーレイ表示」という欄があります。`body`にチェックを入れましょう。すると次のように表示されます。

![](grid_overlay.webp)
縦5行・横6列のグリッド（格子）が引かれていることが確認できます。

[^69]: 完成したCSSを使ってスクリーンショットを掲載しています。今はこのように見えないですが、もうすぐ見えるようになりますので、安心してください。


先ほどチェックを入れた `body` の下に「グリッドの表示設定」という欄があります。「線番号を表示」にチェックを入れると、1番上の線が1、次の線が2、...、一番下の線が7、一番左端の線が1、次の線が2、... 右端の線が7 というように線番号が表示されます。（縦5行・横6列ですので、行や列の数より線の数は一つ多くなります。）

また、「領域名を表示」にチェックを入れると、縦5行・横6列のグリッド（格子）にどのように名前が付けられているのか、領域名を確認することができます。

![](cssgrid.webp)
CSSグリッドレイアウトには、大きくわけて、

1. 行と列の線番号で指定する方法
1. 領域名で指定する方法

があります。

と、指定することができます。

また、領域名を使うと、

* `title`領域に「じゃんけんゲーム」という文字を配置する。

と、指定することができます。

どちらでも全く同じに配置できますが、本書では、より分かりやすい**「領域名で指定する方法」**で配置していきます。

### ヘッダーの装飾

ページ全体のレイアウト枠を決めましたので、続いてヘッダーの装飾を行っていきます。先に紹介したCSSカスタムプロパティで、和の色の指定をしています。
そして、書体を大きくして、色を付けて、影も付け、中央揃えにします。

HTMLとCSSがどのように対応しているかも確認しましょう。
HTMLの`<header>`要素の中に`<h1>`要素はあります。ですので、CSSでも`header`の中で、さらに`h1{}`と記述することで`h1`要素の装飾を行っています。

```html:index.html
<!-- header タグは、サイト名などの表示に用います。 -->
<header>
  <!-- h1 タグは、大見出しを表すタグです。 -->
  <h1>じゃんけんゲーム</h1>
</header>
```

```css:janken.css
/* ヘッダー
---------------------------------------------------------------------*/
header {                      /* header 要素を対象に */
  grid-area: title;           /* title領域に配置する */
  h1 {                        /* header 内の h1 (大見出し)を対象に */
    font-size: 40px;          /* 書体の大きさは40pxにする */
    color: var(--kurohairo);  /* 文字色を黒羽色にする */
    text-shadow: 3px 4px 5px var(--nibiiro); /* 鈍色の影を付ける */
    text-align: center;       /* 文字は中央揃えにする */
  }
}
```

コンピュータの手を表示している部分です。今は「グー」の絵が表示されていますが、後ほどJavaScriptを用いて「チョキ」や「パー」の絵にも変更されるようにします。

HTMLとCSSがどのように対応しているかも確認しましょう。
HTMLの`<figure>`要素の中に`<img>`要素はあります。ですので、CSSでも`figure`の中で、さらに`img{}`と記述することで`img`要素の装飾を行っています。先ほどと全く同様ですね。

```html:index.html
<!-- figure タグは 図版(figure)を示すタグです。 -->

  <img id="hand" src="images/guu.webp" alt="グー">

```

```css:janken.css
/* コンピュータの手を対象に */
figure {
  grid-area: computer;  /* computer領域に配置する */
  img {
    inline-size: 100%;  /* 幅を親要素の幅にする */
  }
}
```


### じゃんけんの説明文の装飾

じゃんけんの説明文に関してですが、HTMLには複数の`<p>`タグがあります。どの`<p>`タグを装飾すべきなのか、区別する必要があります。そのために用いられるのが`class属性`です。`HTML`中に`<p class="note">`と`class属性`を`note` [^70]と設定することで、 `CSS`から、他の`<p>`タグと区別できるようになります。

[^70]: `note`と名前を付けましたが、名称は任意です。自身にとって分かりやすいクラス名を付与してください。

```html:index.html
<!-- p は 段落(paragraph)を表す際に用いるタグです。 -->
<!-- 複数のpタグがあるので、class属性をnoteとして、区別します。-->
<p class="note">
  <!-- br タグを入れると、途中で改行できます。 -->
  みんな知っているじゃんけん<br>
  「開始」ボタンを押すと始まるよ<br>
  グー・チョキ・パー どれかを押してね
</p>
```

`CSS`では、 `**.**note`と書くことで、`<p class="note">` 要素の装飾ができます。

```css:janken.css
/* じゃんけんの説明文を対象に */
.note {
  grid-area: note;      /* computer領域に配置する */
  text-align: center;   /* 文字は中央揃えにする */
}
```


### ボタン要素の装飾

`ID属性`とは、`HTML`要素に固有の名前を付けること為に用いる属性です。同じページ内で重複して使用することはできない(一意である必要がある)という特徴があります。CSSで特定の要素を装飾したり、JavaScriptで特定の要素を操作しやすくなります。

ここでは、じゃんけんの「グー」のボタンに `<button id="guu">` というように、ID属性を付けて、CSSで配置や装飾を行っていきます。

```html:index.html
<!-- プレイヤーの手を選ぶ為のボタンです。 -->
<!-- button タグを使って、画面に「グーチョキパー」のボタンを表示します。
     JavaScriptで操作しやすいようID属性をguuに、値を0にします。 -->
<button id="guu"   value="0" title="グー"></button>
<button id="choki" value="1" title="チョキ"></button>
<button id="paa"   value="2" title="パー"></button>
```

グー、チョキ、パー の三つのボタンは `<button>`要素であることは共通しているので、まずまとめて設定を行います。

```css:janken.css
/* button要素を対象に */
button {
  aspect-ratio: 1 / 1;            /* 横縦比は１対１にします */
  inline-size: 100%;              /* 幅は親要素の幅にする */
  background-size: contain;       /* 背景画像が全て含むようにする */
  background-repeat: no-repeat;   /* 背景画像を繰り返さないようにする */
  border-radius: 15px;            /* 角は 半径(radius) 15pxで丸くする */
  cursor: pointer;                /* カーソルの形状を手のマークにする */
  border: none;                   /* 枠線は無しにする */
}
```

続けて、それぞれ個別のボタンについて、装飾を行います。
`CSS`では、 `**#**guu`と書くことで、`<button id="guu">` 要素の装飾ができます。

```css:janken.css
/* グー、チョキ、パー 各ボタンの配置と背景色と背景画像を指定 */
#guu   {
  grid-area: guu;                                  /* guu領域に配置する */
  background-color: var(--amairo);                 /* 背景色は天色にする */
  background-image: url('images/player_guu.webp'); /* 背景画像はグーにする */
}
#choki {
  grid-area: choki;
  background-color: var(--nanohanairo);
  background-image: url('images/player_choki.webp');
}
#paa   {
  grid-area: paa;
  background-color: var(--kyohiiro);
  background-image: url('images/player_paa.webp');
}
```


### 得点表示の装飾

```html:index.html
<!-- 得点表示です。 -->
<!-- 複数のpタグがあるので、class属性をscoreとして、区別します -->
<p class="score">
  <!-- JavaScriptから操作しやすいよう、id属性をwinとloseにします。 -->
  <span id="win">0</span> 勝
  <span id="lose">0</span> 敗
</p>
```

```css:janken.css
/* 得点表示の設定 */
.score {
  grid-area: score;  /* score領域に配置する */
  font-size: 24px;   /* 書体の大きさは24pxにする */
}
```


### 開始ボタンの装飾

```html:index.html
<!-- button タグを使って、画面に「開始」ボタンを表示します。
     このボタンにも、id="play" と ID を付与しておきます。 -->
<button id="play">開始</button>
```

```css:janken.css
/* 開始ボタンの設定 */
#play  {
  grid-area: play;                      /* play領域に配置する */
  background-color: var(--utsushiiro);  /* 背景色は 移色にする */
  color: var(--sakurairo);              /* 文字の色は 桜色にする */
  font-size: 24px;                      /* 書体の大きさは24pxにする */
  aspect-ratio: 5 / 1;                  /* 横縦比を横5対縦1にする */
}
```
