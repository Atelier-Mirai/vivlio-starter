# 初めてのCSS

:::{.chapter-lead}
ウェブサイトの意匠・装飾を整えるための`CSS(Cascading Style Sheets)`について学びます。CSS は HTML の各要素を選択し、色や大きさ、配置など様々な装飾を記述する為の言語です。

この章では、前の章で作成した `wave-studio` のトップページ `index.html` を中心に、そこから切り出した小さな HTML 文書も交えながら、CSS の基本とレイアウトの考え方を学びます。  
:::



## CSSの基本

それでは飾り付けを行っていきましょう。はじめに小さな HTML 文書で CSS の書き方になじみ、そのあとで `index.html` に色とレイアウトを重ねていきます。


### 練習用のHTMLを準備する

練習用の HTML として、次のような `study.html` を作りましょう。

**▼study.html**

```html
<!DOCTYPE html>
<html>
  <!-- head には、文書に関する設定事項を書きます。 -->
  <head>
    <!-- 文字コードは utf-8 を使います -->
    <meta charset="utf-8">
    <!-- このページのタイトルを示すタグで、ブラウザのタブに表示されます。-->
    <title>ウェブサイトを創る</title>
    <!-- iPhoneで見た際にレイアウトを整えるための設定です -->
    <meta name="viewport" content="width=device-width">
    <!-- スタイルシートを読み込み、サイトを飾り付けします。-->
    <link rel="stylesheet" href="style.css">
  </head>

  <!-- body タグには、サイトのコンテンツ（内容・出し物）を記述します。
       全ての要素は このbodyタグの中に書きます。-->
  <body>
    <!-- h1 タグは、大見出しを表すタグです。他に 中見出しを表す h2 タグや
         小見出しを表す h3 タグなど h6 タグまであります。 -->
    <h1>ウェブサイトを創る</h1>

    <!-- p は 段落(paragraph)を表す際に用いるタグです -->
    <p>ウェブサイトを作成するためには、次の三つの道具が必要です。</p>

    <!-- ul は 順序付き項目(Ordered List)に用いるタグです-->
    <ol>
      <!-- li は リスト項目(List Item)を示す際に用いるタグです -->
      <li>コンピュータ (Mac / Linux / Windows など)</li>
      <li>エディタ (Windsurf / Zed / VS Codeなど)</li>
      <li>ブラウザ (Firefox, Safariなど)</li>
    </ol>

    <p>ウェブサイトの作成には、次の三つの言語が良く使われます。</p>
    <!-- ul は 箇条書き項目(Unordered List)に用いるタグです-->
    <ul>
      <li>HTMLは、「文書構造」の為の言語です。</li>
      <li>CSSは、「意匠・装飾」の為の言語です。</li>
      <li>JavaScriptは、「相互作用・動き」の為の言語です。</li>
    </ul>
  </body>
</html>
```

`<!-- -->`は、コメント(備忘録、覚え書き、説明書き)です。省略しても構いません。


  入力が終わりましたら、作成した `study.html` をブラウザで表示させてみましょう。

  ### スタイルシートの読み込み

スタイルシートは、HTML 本文とは別に、文字の色や大きさ、余白や枠線など「見た目のルール」だけをまとめて書いておくためのファイルです。
それでは、ここからは、自分で色をつけたり枠をつけたりなど、装飾を行って参りましょう。

`<head>`タグの中に書いた次の行で、どのスタイルシートを読み込むのかを指定します。

```html
<!-- スタイルシートを読み込み、サイトを飾り付けします。-->
<link rel="stylesheet" href="style.css">
```

このHTML文書の装飾用に、どのスタイルシートを読み込むのかを指定しています。`href="style.css"` と書くことで、`study.html`と同じディレクトリ(フォルダ)内にある `style.css` に書かれている装飾が行われることになります。

ここからは、ブラウザまかせの標準の装飾だけでなく、自分でCSSを書いて見た目を変えていきます。
CSS では、HTML のどの要素に、どのような見た目のルールを適用するかを記述します。
どの要素を対象にするかを指定する部分を「セレクター」、色や余白などの具体的な指定を書いていく部分を「ルール」と呼びます。

まずは、段落 `p` 要素を赤色にするシンプルな例から見てみましょう。

### CSSで段落を赤色にする

### CSS での コメント の書き方

CSS にも コメント(備忘録、覚え書き、説明書き) を書くことが出来ます。CSS でのコメントは `/* コメント */` のように書きます。先ほどの例にコメントをつけると次のようになります。

**▼style.css**

```css
/* p 要素を選択して、装飾を施す */
p {
  color: red; /* 文字色を赤にする */
}
```

`/* */` は、手入力することもできますが、 Windsurf などのエディタでは、`Comannd + /` でも入力できます。

通常、一行一行にコメントを施すことはありませんが、理解の手助けになればとの観点から記述しました。

CSSでは色や文字の大きさ、配置等、様々な装飾を行うことから、とても長くなります。要所要所にコメントを施すことで、見やすいCSSとなるでしょう。

### CSS 規則セットの構造

:::{.sideimage-right}
![](css-declaration.png){width=50%}

赤い段落テキストの CSS コードを分解して、その仕組みを理解してみましょう。

全体の構造は規則セットと呼びます (単に規則とも呼ばれます)。

それぞれの部分の名前にも注意してください。

- セレクタ (Selector)  
  これは規則セットの先頭にある HTML 要素名です。これはスタイルを設定する要素 (この例では `<p>`要素) を定義(選択・指定)します。別の要素をスタイル付けするには、セレクタを変更してください。

- 宣言 (Declaration)  
  `color: red;` のような単一の規則です。これは要素のプロパティのうち、スタイル付けしたいものを指定します。

- プロパティ (Property)  
  これらは、 HTML 要素をスタイル付けするための方法です。 (この例では、 `color` は `<p>` 要素のプロパティです。) CSS では、規則の中で影響を与えたいプロパティを選択します。

- プロパティ値 (Property value)  
  プロパティの右側には `:(コロン)` の後に プロパティ値 があります。与えられたプロパティの多くの外観から 一つを選択します。 (例えば、 `color` の値は `red` 以外にもたくさんあります。)

構文の他の重要な部分に注意してください。

* セレクタを除き、それぞれの規則セットは中括弧 (`{}`) で囲む必要があります。
* それぞれの宣言内では、コロン (`:`) を使用してプロパティと値を分離する必要があります。
* それぞれの規則セット内でセミコロン (`;`) を使用して、それぞれの宣言と次の規則を区切る必要があります。

一つの規則セットで複数のプロパティ値を変更するには、次のようにセミコロンで区切って書いてください。

**▼style.css**

```css
/* p 要素を選択して、装飾を施す */
p {
  color: red;               /* 文字色を赤にする */
  width: 480px;             /* 幅を指定する */
  border: solid 1px black;  /* 実線で1pxの黒い枠線を付ける */
}
```

:::{.sideimage-right}
![](css-sample2.png){width=50%}

`style.css` に上のコードを記述し、
ブラウザを再読み込みすると、右のようになります。とても読みやすくなりました。

段落の部分が赤くなるほか、幅が480pxになる、黒い枠で囲まれるようになります。

### 複数の要素を選択する

**▼style.css**

```css
/* p 要素 と li 要素 と h1 要素を選択して、装飾を施す */

}
```

**▼style.css**


### フォントとテキスト

色の指定の他、文字の大きさや配置、行間、文字間の調整も行えます。

HTML 本文内にテキストを配置する要素 `<h1>, <li>, <p>` の書体(フォント)の大きさを設定します。また、見出しを中央揃えにしたり、行の高さや文字の間隔などの設定を行い、本文を読みやすくします。

**▼style.css**

```css
/* h1 要素を選択して、装飾を施す */
h1 {
  font-size: 36px;      /* 書体の大きさを指定します */
  text-align: center;   /* 文字は中央揃えにします */
}

/* p1 要素 と li 要素を選択して、装飾を施す */
p, li {
  font-size: 16px;      /* 書体の大きさを指定します */
  line-height: 2;       /* 行の高さを 2 行分にします */
  letter-spacing: 1px;  /* 文字の間隔を 1px 空けます */
}
```

![](css-sample5.png){width=50%}

`style.css` に上のコードを記述し、
ブラウザを再読み込みすると、右のようになります。
書体の大きさや、文字の配置が変更されていることなどを確認してください。
:::

### さまざまな種類のセレクタ

セレクタにはさまざまな種類があります。上記の例では、要素セレクタを使用しており、特定の種類の要素をすべて選択しています。しかし、もっと特定の要素を選択することもできます。ここでは、より一般的なセレクタの種類を紹介します。 [^118]

[^118]: 他にも様々なセレクターがあります。詳しくは、 MDN の [セレクターガイド](https://developer.mozilla.org/ja/docs/Learn/CSS/Building_blocks/Selectors)をご覧ください。

|セレクター名|選択するもの|例|
|---|---|---|
|要素セレクタ|指定された型のすべてのHTML要素|`p`要素（例: `<p>`）を選択|
|IDセレクタ|指定されたIDを持つページ上の要素。1つのHTMLページ内で各id値は一意でなければなりません。|`#my-id`（例: `<p id="my-id">`や`<a id="my-id">`）を選択|
|クラスセレクタ|指定されたクラスを持つページ上の要素。同じクラスを持つ複数の要素が1ページに現れることがあります。|`.my-class`（例: `<p class="my-class">`や`<a class="my-class">`）を選択|

### IDセレクタを使う

それでは、早速 ID セレクタを使ってみましょう。

HTML を次のように、書き換えましょう。
箇条書き項目のそれぞれに `id="html"` `id="css"` `id="javascript"` と名前({識別子|しきべつし})を与えています。(ID属性)

名前({識別子|しきべつし}) は、同じHTML文書内で「唯一」である必要があります。

```html
<ul>
  <li id="html">HTMLは、「文書構造」の為の言語です。</li>
  <li id="css">CSSは、「意匠・装飾」の為の言語です。</li>
  <li id="javascript">JavaScriptは、「相互作用・動き」の為の言語です。</li>
</ul>
```

HTML内に記述した ID属性 `id="html"` を、CSSから参照するには、`#html` のように書きます。今まで書いてきた `style.css` に、次のコードを書き加えてみましょう。

**▼style.css**

```css
/* ID属性が html である要素(id="html")を選択して、装飾を施す */
#html {
  background: orange;   /* 背景色を指定します */
}

#css {
  background: cyan;
}

#javascript {
  background: yellow;
}
```

![](css-sample6.png){width=50%}

ブラウザを再読み込みすると、右のように、それぞれの言語に応じた背景色も付き、美しくなりました。
:::

### クラスセレクタを使う

次に、クラスセレクタを使ってみましょう。

HTML を次のように、書き換えましょう。

順序付き項目のそれぞれに `class="hardware"` `class="software"` `id="software"` とクラス名を与えています。(クラス属性)

クラス名は、同種の要素として「分類」する際に用います。ID属性とは異なり、同じHTML文書内で複数あっても差し支えありません。コンピュータは「ハードウェア」ですが、エディタやブラウザは「ソフトウェア」ですので、同じクラス名を付けています。

<ol>
<li class="hardware">コンピュータ (Mac / Linux / Windows など)</li>
<li class="software">エディタ (Windsurf / Zed / VS Codeなど)</li>
<li class="software">ブラウザ (Firefox, Safariなど)</li>
</ol>

そして、`style.css` に次のコードを書き加えます。

**▼style.css**

```css
/* class属性が hardware である要素(class="hardware")を選択して、装飾を施す */
.hardware {
  border-bottom: solid 3px lime;      /* 下の枠線を 実線で引きます */
}

.software {
  border-bottom: double 3px magenta;  /* 下の枠線を 二重線で引きます */
}
```

:::{.sideimage-right}
![](css-sample7.png){width=50%}

ブラウザを再読み込みすると、それぞれの種類に応じた背景色も付き、さらに美しくなりました。
:::

「IDセレクタ」と「クラスセレクタ」との違いを確認しましょう。

* IDセレクタを用いると、ページ内に存在する唯一の要素を選択して装飾できます。
* クラスセレクタを用いると、ページ内に存在する同分類の要素を選択して装飾できます。

クラスセレクタはとても良く出てきます。活用して、素敵なサイトを作っていきましょう。

### CSS の優先順位

先ほどのコードの中から一部分を取り上げてみましょう。

`h1`要素に対して、文字の大きさや配置を指定するコードです。

**▼style.css**

```css
/* h1 要素を選択して、装飾を施す */
h1 {
  font-size: 36px;      /* 書体の大きさを指定します */
  text-align: center;   /* 文字は中央揃えにします */
}
```

`h1`要素に関する装飾を、同じ規則セットの中に書いていますが、次のように分けて書いても同じになります。

**▼style.css**

```css
/* h1 要素を選択して、装飾を施す */
h1 {
  font-size: 36px;      /* 書体の大きさを指定します */
}

/* h1 要素を選択して、装飾を施す */
h1 {
  text-align: center;   /* 文字は中央揃えにします */
}
```

それでは、同じプロパティの指定を二度書くとどうなるのでしょう。
つまり、次のようにです。

**▼style.css**

```css
/* h1 要素を選択して、装飾を施す */
h1 {
  font-size: 36px;      /* 書体の大きさを指定します */
}

/* h1 要素を選択して、装飾を施す */
h1 {
  font-size: 18px;      /* 書体の大きさを指定します */
}
```

:::{.sideimage-right}
![](css-sample8.png){width=50%}

結果は、右のように文字が小さくなりました。

同じ適用先に同じプロパティを複数指定した場合、後から記述した設定が優先され、先の記述を上書きします。
:::

他の優先順位についても簡単に整理しておきましょう。

- `li` などの単一タグセレクタより、`ol li` のような複数タグセレクタの方が強い
- 複数タグセレクタより、クラスセレクタ（`.item` など）の方が強い
- クラスセレクタより、IDセレクタ（`#item` など）の方が強い

より詳しいルールや計算方法は、MDN の
[詳細度](https://developer.mozilla.org/ja/docs/Web/CSS/Specificity) や
[CSS カスケード入門](https://developer.mozilla.org/ja/docs/Web/CSS/Cascade)
を参照してください。

## CSSの色指定

### RGBカラー

:::{.sideimage-right}
CSS では、光の三原色である赤 (Red)・緑 (Green)・青 (Blue) を組み合わせた **RGBカラー** が主流です。

![](rgb_color.svg)

光の三原色は「赤 Red」「緑 Green」「青 Blue」です。  
これらを重ね合わせることで、さまざまな色が得られます(加法混色)。

- 赤 + 緑 → 黄 (Yellow)
- 緑 + 青 → 青緑 (Cyan)
- 青 + 赤 → 赤紫 (Magenta)
- 赤 + 緑 + 青 → 白（White）

:::

一方、色の三原色と呼ばれる「黄・赤紫・青緑」を重ねると暗くなり、最終的には黒に近づきます（減法混色）。  
光の三原色と色の三原色は、美しい相補関係にあります。

光の三原色 赤(R)・緑(G)・青(B) の明るさを「消灯」「点灯」の 2 段階だけで組み合わせると $2^3 = 8$ 色ですが、各色を $0$〜$255$ の $256$ 段階にすると、組み合わせは $256^3 = 16,777,216$ 通りになり、非常に多くの色を表現できます。

CSS では、これらを `#RRGGBB` 形式の 16 進数で指定します。たとえば、

- 赤 : `#ff0000`
- 緑 : `#00ff00`
- 青 : `#0000ff`
- 赤 + 緑 の黄 : `#ffff00`

のように表現します。

公式サイト MDN の [color](https://developer.mozilla.org/ja/docs/Web/CSS/color) や  
[CSS を使った HTML の要素への色の適用](https://developer.mozilla.org/ja/docs/Web/CSS/CSS_Colors/Applying_color) に、より詳しい解説があります。

### CSSカラーネーム

良く使う色149色(内別名定義2色)には「CSSカラーネーム」として名前が付けられています。
その為、`color: #ff0000; ` に代えて `color: red;` と書いても、文字色を赤にすることが出来ます。
CSSカラーネームの一部を [原色大辞典](https://www.colordic.org)より抜粋、掲載します。

### 半透明

CSS では、色に「どのくらい透けて見えるか」を表す **透明度（アルファ値）** を指定できます。

16進数では `#RRGGBBAA` のように、最後の `AA` が透明度を表します。  
たとえば `#ff000080` は、赤を 50% 透明にした色です。

同じことを関数形式で書く場合は、`rgb()` の後ろに `/` で透明度を指定します。  
たとえば `rgb(255 0 0 / 50%)` も、赤を 50% 透明にした色を意味します。
