## 初めてのJavaScript

JavaScriptのプログラムを書き始めていきましょう。JavaScript は、HTMLから呼び出される形で良く利用されます。HTMLからJavaScriptを呼び出すためには次のように書きます。

**▼index.html**

```
<!-- DOCTYPE(ドクタイプ)宣言。歴史的経緯から一番先頭に書きます。-->
<!DOCTYPE html>
<!-- HTMLの始まり。lang="ja"と書き、日本語のサイトであることを示します -->
<html lang="ja">
  <!-- head には、文書に関する設定事項を書きます。 -->
  <head>

    (略)

    <!-- スタイルシートを読み込み、サイトを飾り付けします。-->
    <link rel="stylesheet" href="janken.css">
    <!-- JavaScriptを読み込み、じゃんけんゲームを動かします。 -->
    **<script src="janken01.js" defer></script>**
  </head>
```

36行目の `<script src="janken01.js" defer></script>` が、`janken01.js` というJavaScriptのファイルを読み込んでいるコードです。

`janken01.js` は、次のように書きましょう。

**▼janken01.js**

```
// じゃんけんの勝ち負けの結果を表示する
alert("あなたの勝ちです!")
```

プログラムを書き終わったら保存しましょう。そしてブラウザを再読み込みします。すると、右のように「あなたの勝ちです!」とメッセージが表示されます。<br>
　おめでとうございます。初めてのJavaScriptは、これで完成です!!

**▼セミコロン有りのコード**

```

```

**▼セミコロン無しのコード**


二種類のコードを掲載いたしました。とてもよく似ているコードで、違いは行末に`;(セミコロン)`が有るのか無いのかだけで、どちらも正しく動きます。

## 変数

プログラミング言語には、文字列や数値などのデータに名前をつけることで、繰り返し利用できるようにする変数という機能があります。「変数」とは「値」が格納できる「箱」のようなものであると想像すると理解しやすいでしょう。

より正確な定義が、[IT用語辞典](https://e-words.jp/w/変数.html) にございますので、確認してみましょう。

> 変数とは、コンピュータプログラムのソースコードなどで、データを一時的に記憶しておくための領域に固有の名前を付けたもの。変数につけた名前を変数名と呼び、記憶されているデータをその変数の値という。データの入れ物のような存在で、プログラム中で複数のデータを扱いたいときや、同じデータを何度も参照したり計算によって変化させたい場合に利用する。
>
>
> 変数をプログラム中で利用するには、これからどんな変数を利用するかを宣言し、値を代入する必要がある。コード中で明示的に宣言しなくても変数を利用できる言語もある。変数に格納された値を利用したいときは、変数名を記述することにより値を参照することができる。

定義ですので、少し難しいかもしれませんが、何度も出てきますので、すぐに使いこなせるようになる事と思います。

では、これから学ぶ JavaScript ではどのように「変数」を使えば良いのでしょうか。JavaScript では、どのような名前の変数を自分が書くプログラムで使うのか、最初に「宣言」する必要があります。変数宣言をする為のキーワードとして `const, let` の2つが用意されていますので、順に見て行きましょう。


:::{.tip}
* `let`   は、「変数」を宣言する時に使います。何度でも値の変更が可能です。
:::

それでは、使って見ましょう。

**▼変数宣言の例**

```
// コンピュータの手として、computer という変数を宣言する
let computer
```

変数の宣言をすると、その変数に値を入れる（代入する）ことができます。

**▼変数に値を入れる例**

```
// コンピュータの手として、computer という変数を宣言する
let computer

// computer という変数に値を入れる
computer = 0
```

変数の宣言と同時にその変数に値を入れる（代入する）こともできます。

**▼変数宣言と、変数への値の代入を同時に行う例**

```
// コンピュータの手として、
// computer という変数を宣言するとともに、
// computer という変数に値を入れる
let computer = 0
```

変数には値を何度でも代入することができます。
次の例は、0, 1, 2 と変化させた例です。

**▼変数への値の代入を何度も行った例**

```
computer = 0
computer = 1
computer = 2
```

また、変数名の宣言は一度だけで十分です。同じ変数名で何度も宣言する必要はありませんし、また出来ませんので、ご注意下さい。

次の例はエラー（誤り）となります。

**▼変数宣言を何度も行った例**

```
let computer
let computer = 0
computer = 1
let computer = 2
```


:::{.note}
**=(イコール)記号について**

`=(イコール)`は、数学ですと、**computerと1が「等しい」**ということを意味する記号ですが、プログラミングの世界では **左の変数に右の値を入れる（代入する）** という記法になります。

初期には「等しい」ことを示す `=`と区別するために別の記号が用いられた時代もありましたが、C言語（シーげんご）やRuby（ルビー）, JavaScript（ジャバスクリプト）など現在主流の多くの言語では「代入する」記号として`=`を用いています。（「等しい」意味では == や === を用います。）

始めのうちは **`computer ← 1**` のことだと、頭の中で置き換えると良いでしょう。

:::


## 定数

それでは、次に「定数」を見て行きましょう。
[IT用語辞典](https://e-words.jp/w/定数.html) には、次のように書かれています。

> 定数とは、コンピュータプログラムのソースコードなどで、ある特定の数値やデータに名前を与えたもの。変数とは異なり、宣言時に決めた値をコードの中で後から変更することができない。
>
> プログラムのコード中で何度も参照される値で、実行時に変更する必要のないものに用いられる。値（数値や文字列）が開発の途中で変更される可能性がある場合や、値が長くて何度も入力すると打ち間違える危険がある場合、値に分かりやすい名前をつけたい場合などに用いられる。

「定数」[^74] を宣言する為には、`const` というキーワードを使います。英語の `constant(一定の、不変の)`に由来しています。定数は一度決めた値を変更することは出来ません。（しようとするとエラーになります。）

[^74]: 厳密には「再代入できない変数」の宣言です。


:::{.tip}
* `const`   は、「定数」を宣言する時に使います。初回のみ値を設定できます。
:::

それでは、早速、使って見ましょう。先の変数の例では、`computer = 0`などと書いていました。`computer`という変数に`0`を代入するという意味ですが、この`0`という数値は一体どういう意味なのか、覚えておくのも大変です。そこで分かりやすい名前を付けたいと思います。

**▼定数宣言の例**

```
// 定数宣言
const GUU   = 0 // グー
const CHOKI = 1 // チョキ
const PAA   = 2 // パー
```

`const GUU` と 定数を宣言すると同時に `= 0` で値を設定しています。

こうして定数を宣言しましたので、プログラム中では次のように使うことが出来ます。

**▼定数を利用した例**

```
// コンピュータの手
let computer = GUU
```

`let computer = GUU`と、プログラムを書くことが出来るようになりました。`let computer = 0` と書くよりも文字数は増えていますが、`0`, `1`, `2` の単なる数字とは異なり、その意味するところが明確で、とても分かりやすくなりました。活用してきましょう。


:::{.note}
**以前使用されていた変数宣言のキーワード**

JavaScript には、`let, const` の他に、`var`という変数宣言の為のキーワードもあります。

`var` は、以前は使われていましたが、数々の不具合があるので、`let, const` が導入されました。

今でも、古いプログラムコードには使われていますが、これから書くときには、`let` または `const` を用いるようにしましょう。

:::


## 命名規則

`computer` という名前の変数や、`GUU` という名前の定数を宣言しましたが、変数や定数にはどのような名前を付けられるのでしょうか。

変数名には アルファベットと一部の記号 `a-z, A-Z, $, _(アンダースコア)` が使えます。これらのアルファベットと一部の記号の後ろに `0-9` までの数字を続けることも出来ます。
また、JavaScript 自身が使う為の単語（予約語と言います）そのものは、変数名や定数名として宣言することは出来ませんが、予約語の一部を含むものは使うことができます。


:::{.tip}
**使える例**

* `let a               // 一文字の変数名も使えます`
* `let computer        // 英単語も使えます`
* `let co**n**puter        // つづりが間違っていても使えます`
* `let player1         // 英単語の後ろに数字を繋げても良いです`
* `let player9876      // 沢山の数字を繋げることも出来ます`
* `let playerName      // 英単語を繋げるときには(慣習的に)先頭を大文字にします`
* `let player**_**name     // (JavaScriptでは一般的ではありませんが) _ も使えます`
* `const GUU           // 定数を宣言する時には(慣習的に)全て大文字にします`
* `const GUU_CHOKI_PAA // いくつかの単語を繋げるときは _ で繋ぎます`<br>

* `let kn**if**e            // if は予約語なので使えませんが、ifを含む単語は使えます`
:::

:::{.tip}
**使えない例**

* `let 123               // 数字そのものを変数名にすることは出来ません`
* `let 20th_century_year // 数字から始まる変数名を使うことも出来ません`
* `let twentieth-century // -(ハイフン)は使えない記号です`
* `let if;               // if は予約語なので使えません。`
:::

## 演算子（えんざんし）

![](operator_five.png)

JavaScriptはプログラミング言語ですので、様々な計算を行うことができます。「**演算子**」とは $7 + 5$ の $+$ や、$18 - 3$ の $-$ のように、どのような計算(演算)を行うかを示すために用いる記号が「**演算子**」です。

足し算には$+$を、引き算には$-$を使うことは、日常慣れ親しんだ算数や数学と同じです。掛け算や割り算は、キーボードに $\times$ や $\div$ の記号がないので、代わりに `*（アスタリスク）`や`/（スラッシュ）`を用います。


### 剰余（じょうよ）演算

「剰余」演算は耳慣れないかもしれませんが、平たく言えば、「余りを求める」演算の事です。

割り算の答のことを商（しょう）(quotient)、余りのことを剰余（じょうよ）(remainder)と言います。

$22 \div 7 = 3 余り 1$ です。プログラムでは次のように書きます。

[centering]  
`quotient  = 22 / 7`<br>
`remainder = 22 % 7`  
[/centering]

剰余演算は周期性の有るものを分類する時に便利です。例えば、ある年の元日(1月1日)が月曜日だったとします。元日から21日経過した日(=1月22日)の曜日を知る為には、

$$
1 \% 7 = 1 (1日は月曜日)
$$


$$
22 \% 7 = 1 (22日も月曜日)
$$


と、7 で割った余りがどちらも1になるので、月曜日と知ることが出来ます。


元日から31日経過した日(=2月1日)の曜日を知るにはどうしたら良いでしょうか。

$$
32 \% 7 = 4 (1は月, 2は火, 3は水, 4は木曜日) <br>
$$


として、木曜日であることを求めることが出来ます。[^75]

[^75]: 年月日から曜日を求める公式として [ツェラーの公式](https://ja.wikipedia.org/wiki/ツェラーの公式) がよく知られています。


じゃんけんの勝ち負けを効果的に判定する方法として、後ほど、「剰余演算」が登場します。お楽しみに。


## 関数

関数とは、ある一連の手続き（文の集まり）を一つの処理としてまとめる機能です。関数を利用することで、同じ処理を毎回書くのではなく、一度定義した関数を呼び出すことで同じ処理を実行できます。
[^76]

[^76]: 例えば、勝ち負けの判定を纏めておくと、使い勝手が良さそうです。

![](houteishiki.jpg)

「関数」といえば、数学で良くある関数を思い浮かべる方もいると思います。
例えば、次のような二次式（放物線）の関数です。

$$
y = f(x) = x^2-4x+3
$$


ある $x$ を与えると、それに対応した値 $y$ を返します。

JavaScriptも、値を返す関数を定義できます。値を返すことが目的ではなく、何か処理を行うことを目的とする関数も作成できます。 [^77]

[^77]: 例えば、前に登場した `alert`関数も、メッセージ表示処理を行うことが目的の関数です。

それでは、JavaScript での関数の書き方を学んでいきましょう。


### 関数の定義

二つの数 `a` と `b` を与えると、その和を返す関数`add`を定義してみましょう。

JavaScriptには、従来型の関数記法とアロー関数という二つの関数の定義方法があります。

約30年前に登場した当時は次のように関数を記述していました。

**▼従来からある関数宣言**

```
function add(a, b) {
  return a + b
}
```

10年ほど前に、より簡潔に記述できるようにと、新しい関数の書き方「**アロー関数**」が導入されました。アロー関数は、関数を表す為の記号 ` => ` が、「矢印(arrow（アロー）)」に見えるのが由来で、次のように書きます。 [^78]

[^78]: [アロー関数式](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Functions/Arrow_functions) には、詳しい説明もございます。ご興味を持った方はお読みになってください。

**▼新しく導入されたアロー関数**

```
const add = (a, b) => {
  return a + b
}
```

関数には、4つの構成要素があります。

1. 関数名 <br>
   関数の名前です。この例では `add` です。
2. 仮引数（かりひきすう）<br>
   関数呼出時に渡された値が入る変数です。この例では `a` と `b` です。複数ある場合には `a, b` のように、`,（カンマ）`で区切ります。(必ず何かの処理を行うなど)仮引数が不要の場合は、書かなくても構いません。
3. 関数の中身 <br>
   関数で行いたい処理は、`{` と `}` で囲んだ中に書きます。
4. 関数の返り値 <br>
   関数を呼び出し、処理を行ったあと、呼び出し元に何か値を返したい場合に記述します。この例では、 `return a + b` と書くことで、`a` と `b` の和を呼び出し元に返しています。


### 関数の使用法

定義した関数は、それを呼び出すことで使うことができます。

```
// 変数aを定義し9を代入
let a =  9
// 変数bを定義し23を代入
let b = 23

// 変数cを定義すると同時に、関数addを呼び出します。
// a と b を足した結果が欲しいので、
// addの仮引数として、a(=9)とb(=23) を渡しています。
// add関数 の内部で処理が実行され、23 が返ってくるので、
// 変数cで受け取ります(変数cに代入します)。
let c = add(a, b)

// 変数cの値を表示させます
console.log(c)
```

console.log() は、値を表示させる関数 [^79] で、JavaScript に始めから備わっている関数です。

[^79]: プログラミング言語の標準的な機能として予め定義されている関数のことを「組み込み関数」と言います。


### 従来型の関数とアロー関数
「変数」とは「値」が格納できる「箱」のようなものです。ですので「9」や「23」といった数値、「September」などの文字列を代入（格納）できるのは、理解しやすいでしょう。

そして、JavaScriptの関数は「第一級オブジェクト」と呼ばれ、関数を変数に代入できる！性質を持ちます。9と23を足し算した結果である32を変数に代入するのではなく、「足し算する」という「機能・働き・関数」そのものを変数に代入できるという、特筆すべき性質を持っています。

ですので、次のように 変数 `add` に 関数 `tashizan`を代入することができます。

```
// 変数 add に 関数tashizanを代入する
const add = function tashizan(a, b) {
  return a + b
}
```

そして使うときには、あたかも変数`add`が関数であるかのように（関数を代入しているのですから全く関数と同じように）利用できます。

```
// 関数の呼び出しと結果の表示
const answer = add(9, 23)
alert(answer)
```

変数 `add` に代入して、「足し算をする」という機能を呼び出せるのであれば、わざわざ 関数に `tashizan` と命名する必要も無いので、名前を省けます（「無名関数」と言います）。

```
// 変数 add に 無名関数を代入する
const add = function (a, b) {
  return a + b
}
```

`function` と毎回書くのも文字が長いので省略して、代わりに アロー関数の記法 ` => ` を ` () ` の次に書きます。

**▼アロー関数の完成**

```
const add = (a, b) => {
  return a + b
}
```

古くから親しまれてきた `function` という従来型の関数宣言ですが、「アロー関数」は、より簡潔に書けるよう様々な利点、機能向上が図られています。使って行きましょう。

## イベントリスナーと関数


### イベントリスナーとは

変数や関数などJavaScriptの基礎を学びました。続けてイベントリスナーについて学びましょう。

JavaScriptには、**イベントリスナー** と呼ばれる仕組みが有り、「何か操作を行った時」に「指定した動作をする」機能が備わっています。

最初に作ったプログラムでは、プレイヤーが何もすることなく、勝利メッセージが表示されました。これではゲームらしくありません。イベントリスナーを使うと、「グー」ボタンを押したときに「勝利メッセージ」を出せるようになります。

**イベント**[^80]、**リスナー**[^81]それぞれの用語の意味を調べてみましょう。


:::{.tip}
イベントは、プログラム内で発生した動作・出来事、またそれらを表現する信号である。
ウェブブラウザにおける「ページが読み込まれた時」「クリック動作」「スクロール操作」などさまざまなイベントがある。
:::

[^80]: 出典：Wikipedia


:::{.tip}
リスナーとは、聞く人、聞き手、聴取者、聴講生、などの意味を持つ英単語。一般の外来語としてはラジオ（局/番組）の聴取者を意味する用法が有名である。

プログラミング分野では、何らかのきっかけに応じて起動されるよう設定された関数やメソッドなどをイベントリスナーと言う。例えば、「マウスがクリックされると起動するよう指定された関数」のことを「マウスクリックを待ち受けるリスナー」といったように呼ぶ。
:::

[^81]: 出典：IT用語辞典

つまり**イベントリスナー**[^82]とは次のようになります。


:::{.tip}
「ページ読み込みやクリック動作など、ウェブページで行われる様々な動作を常時起動し待ち受けて(聴取し続けて)、イベントが起きた時に指定された処理を行う関数のこと」
:::

[^82]: 参考：フロントエンドエンジニアを目指す! JavaScript講座(9)イベントリスナーを使用する

## 繰り返し処理


### `while`文

三回、勝利メッセージを表示したいときは、どのようにすれば良いでしょうか。

```
// じゃんけんの勝ち負けの結果を表示する関数
const jankenHandler = (event) => {
  alert("あなたの勝ちです!")
  alert("あなたの勝ちです!")
  alert("あなたの勝ちです!")
}
```

このように三回書いたらできます。それでは百回表示させたいときはどうでしょうか。エディタのコピー機能を使うこともできますが、それでも百回書くのは大変です。

プログラミング言語には、同じ処理を繰り返したいときの為の専用の構文(書き方)が用意されています。

先ほど作成した`janken02.js`を複製して`janken03.js`を作りましょう。呼び出し元となる `index.html` のコードも`<script src="janken03.js" defer></script>` と更新しましょう。

準備ができたら、`while文` を使い、次のように書いてみましょう。

**▼janken03.js**

```
// じゃんけんの勝ち負けの結果を表示する関数
const jankenHandler = (event) => {
  // 繰り返し処理
  let i = 0
  while (i < 3) {
    alert("あなたの勝ちです!")
    i = i + 1
  }
}

// イベントリスナの設定
(以下同じ)
```

`while`文とは、条件式が「真（しん）」の間（成り立つ間）、繰り返し処理を行う命令文（制御構造）で、このコードは次のように動作します。


:::{.tip}
1. 変数`i`を`0`で初期化します。
2. `while`文の条件式 `(i < 3)` を評価します(条件が成り立つかどうか判断します)。最初は`0 < 3` なので条件は真（しん）になります(成り立ちます)。
3. 条件式が真なので、 `alert("あなたの勝ちです!")` が実行され、ダイアログボックスが表示されます。
4. その後、 `i = i + 1` が実行され、`i`の値が`1`になります。
5. `while`文の条件式 `(i < 3)` を再度評価します。今度も `i < 3` なので条件は真になります。
6. 手順2-5が繰り返されます。
7. 4回目のループで、`i`の値が`3`になり、条件式 `(i < 3)` が偽（ぎ）になる(成立しない)ため、`while`文の処理が終了します。
:::

変数`i`を使って、繰り返し回数を数える [^85]ことにより、「あなたの勝ちです!」というダイアログを3回表示することができました。

[^85]: 繰り返し回数を数える変数のことを、**カウンタ変数**と言います。

因（ちな）みに`i < 3` と書かれているところを、`i < 100` と書き換えると、百回表示させることもできます。[^86]

[^86]: [アラートループ事件](https://ja.wikipedia.org/wiki/アラートループ事件) 兵庫県警によりウイルス罪として女子中学生ら五人が捕えられた誤認・冤罪事件。JavaScriptの生みの親であるブレンダン・アイクは「Chromeの初版よりも10年も前にリリースされたNetscape 4でもユーザーがループするJavaScriptを切ることができた。」「この事件の公判で専門家証人になるつもりでいる。これはウイルスなどではなく、犯罪扱いされるべきではない。」とツイートした。


### `for`文

回数を指定して繰り返す処理は良く行われます。そこで`for`文と呼ばれる専用の構文も用意されています。

**▼for文**

```
for (初期化式; 条件式; 更新式) {
  処理
}
```

`for`文は次のように動作します。

- 1. 初期化式を最初に1回だけ実行します。
- 2. 条件式が評価(成り立つかどうか判断)します。

* 真の場合(成り立つ場合)は、処理を実行します。
* 偽の場合(成り立たない場合)は、繰り返し処理を終了します。

- 3. 更新式を実行します。
- 4. 2に戻って繰り返します。

**▼janken03.js**

```
// じゃんけんの勝ち負けの結果を表示する関数
const jankenHandler = (event) => {
  // 繰り返し処理
  for(let i = 0; i < 3; i++) {
    alert("あなたの勝ちです")
  }
}

(以下同じ)
```


:::{.note}
**while文とfor文の使い分け**

繰り返しの為の構文には、基本となる`while`文と、便利に書ける`for`文の二種類があります。次のような基準で使い分けると良いでしょう。


**`while**文を使うと良い場合`

* 繰り返し回数が事前に分かっていない場合
* 条件式が複雑で、単純な初期化・更新式では表現できない場合
* 条件式が真の間、処理を続ける必要がある場合


**`for**文を使うと良い場合`

* 繰り返し回数が事前に分かっている場合
* 初期化、条件式、更新式が簡単に表現できる場合
* 配列やリストの要素を順番に処理する場合

:::


## 条件分岐

じゃんけんは、勝つこともあれば、負けることもあります。条件分岐を使うと、勝敗によって行う処理を変更できます。


### `if（イフ）文`

`if（イフ）文` は条件分岐の為に用意されている構文です。

条件式の評価結果が `true（トゥルー）` (真（しん）・成り立つ)であるならば、波括弧{ \}内に書かれた処理が実行されます。

```
if (条件式) {
  条件式が真の場合(成り立つ場合)の処理
}
```

条件式が`false（フォールス）` (偽（ぎ）・成り立たない) 場合の処理も行いたい場合には次のように書きます。

```
if (条件式) {
  条件式が真の場合(成り立つ場合)の処理
} else {
  条件式が偽の場合(成り立たない場合)の処理
}
```

複数の条件式で評価することもできます。

```
if (条件式1) {
  条件式1が真の場合(成り立つ場合)の処理
} else if (条件式2) {
  条件式2が真の場合(成り立つ場合)の処理
} else if (条件式3) {
  条件式3が真の場合(成り立つ場合)の処理
} else {
  いずれの条件式も偽の場合(成り立たない場合)の処理
}
```

それでは `if文` を使って行きましょう。JavaScriptファイルを複製して`janken04.js`を作りましょう。呼び出し元の `index.html` も `<script src="janken04.js" defer></script>)` と更新します。

### 等価演算子と比較演算子

さてこの例では、プレイヤーの手がグー(0)に等しいか、チョキ(1)に等しいか、それ以外(パー)かで、条件分岐しました。この判定に用いた演算子が「等価演算子」です。また大小関係を判定する際に用いる「比較演算子」もあります。 [^89]

[^89]: 10勝したらチャンピオン認定するなど、プログラムを発展させる際に役立ててください。


**等価演算子と比較演算子**

| コード | 意味 |
| --- | --- |
| `win === 0` | 0に等しいか |
| `win !== 0` | 0に等しくないか |
| `win < 0` | 0未満 |
| `win <= 0` | 0以下 |
| `win > 0` | 0超 |
| `win >= 0` | 0以上 |


### 論理演算子

真偽値の評価（条件式が成り立っているか否かの判断）を行う為の演算子として、次のような論理演算子もあります。


#### 論理否定(NOT)

真偽値を反転させる演算子です。

[centering]  
NOT真 = 偽 <br>
NOT偽 = 真  
[/centering]

となります。演算子の記号は `!(エクスクラメーションマーク)` を用います。

例えば数字の`10`の前に`-(マイナス)`を付けて`-10`とすることで、正の数が負の数になるように、`if(!真)`と書けば`if(偽)`と書いたのと同じ（＝条件が成り立っていない場合に）となります。

[centering]  
例: 「あいこ」でない場合のみ、勝敗を更新する。 <br>
`if(!draw){ 勝敗を更新する }`  
[/centering]


#### 論理積(AND)

2つの真偽値が両方とも真の場合にのみ真となる演算子です。

[centering]  
真 AND 真 = 真 <br>
真 AND 偽 = 偽 <br>
偽 AND 真 = 偽 <br>
偽 AND 偽 = 偽  
[/centering]

となります。演算子の記号は `&&(アンパサンド アンパサンド)` を用います。

[centering]  
例: 10勝以上、かつ0敗の場合にのみ、「じゃんけん覇者」と表示する。 <br>
`if(win >= 10 && lose === 0){ じゃんけん覇者}`  
[/centering]


#### 論理和(OR)

2つの真偽値のいずれかが真の場合に真となる演算子です。

[centering]  
真 OR 真 = 真<br>
真 OR 偽 = 真<br>
偽 OR 真 = 真<br>
偽 OR 偽 = 偽  
[/centering]

となります。
演算子の記号は `||(バーティカルバー バーティカルバー)` を用います。

[centering]  
例: 10勝以上、または0敗の場合に、「じゃんけん猛者」と表示する。 <br>
`if(win >= 10 || lose === 0){ じゃんけん猛者 }`  
[/centering]

### 定数宣言

解説です。まず冒頭に書かれた3行目から5行目では「定数」を宣言しています。

「グー」は`0`、「チョキ」は`1`、「パー」は`2` と数値化することにより、プログラムでは扱いやすくなりますが、プログラムを書く人、プログラマにとっては数値の意味するところを記憶しておかねばならず扱いにくいものです。そのため数値に名前を付けることにします。これが「定数」で、慣習的に定数名は全て大文字で書きます。

定数宣言したので、8行目、18行目、22行目では、数値の代わりに`GUU`や`CHOKI`という定数名で書き直しています。

ぐっと分かりやすくなりましたね。


### プレイヤーの手の取得

`index.html` で次にように書かれていたことを思い出してください。

**▼index.html**

```
<button id="guu"   value="0" title="グー"></button>
<button id="choki" value="1" title="チョキ"></button>
<button id="paa"   value="2" title="パー"></button>
```

`HTML`側で「グー」は`0`、「チョキ」は`1`、「パー」は`2` と `value`属性に値を設定しています。

この`value`属性の値を`JavaScript`で取得すれば、どのボタンが押されたのか分かります。それが次のコードです。

```
// じゃんけんの勝ち負けの結果を表示する関数
const jankenHandler = (event) => {
  // プレイヤーの手の取得
  const player = Number(event.target.value)
```

11行目で `jankenHandler`関数に引数`event`が渡されています。`event.target`と書くと、イベントの呼び出し元の要素を取得することができます。つまり、グー、チョキ、パー、どのボタンを押したことにより`jankenHandler`が呼ばれたのか、それを知ることができます。

`event.target`で、イベントの呼び出し元の要素を取得できたので、これから`value`属性値を得るには、`value（バリュー）`メソッドを使います。

つまり `event.target.value` と書くことで `<button id="guu" value="0"></button>` と書かれていた `value` 属性の値 `"0"` を取得することができます。

`Number`は、文字列としての`"0"`を整数値 `0`に変換する関数です[^90]。数値`0`に変換したので、`let player = `と書いてプレイヤーの手を保持する変数 `player` を宣言し、代入しています。

以上で、プレイヤーが入力した（ボタンを押した）手を取得できました。

[^90]: ちょうど、漢数字の `〇`と、数値の `0` が異なるように、一般にプログラミング言語では、文字列としての `"0"` と、整数値としての `0` とは異なります。

### 乱数を使う

コンピュータの手は「`computer = 0(GUU)`」でしたので、プレイヤーは「`2(PAA)`」を出せば、必ず勝てます。
コンピュータも無作為に「グー」「チョキ」「バー」の手を出すようにしましょう。

そのためには、「乱数」と呼ばれる機能を使います。
JavaScriptには `Math.random()` と呼ばれる関数が用意されています。
`Math.random()` を呼ぶと、0から1未満の浮動小数点数を返します。
[^91]

[^91]: 表示は一例で、「乱数」なので実行するたびに値は変わります。

コンソール画面に `Math.random()` と入力してみましょう。

```
Math.random()
0.9200533064823014
0.5017996980638613
0.15088182883224843
```

わたくしたちが欲しいのは、0, 1, 2 の3つの数ですから、3倍すれば良さそうです。

```
Math.random() * 3
2.760159919446904
1.5053990941915838
0.4526454864967453
```

小数点以下は不要ですから、切り捨てましょう。
JavaScriptには `Math.floor()` と呼ばれる切り捨ての為の関数が用意されています。 [^92]

[^92]: `Math.floor()`は床関数と呼ばれます。 `Math.ceil()` は天井関数と呼ばれ 切り上げを、 `Math.round()`は丸め関数と呼ばれ 四捨五入を行います。

```
Math.floor(Math.random() * 3)
2
1
0
```

ですので、次のように書けば、 `computer` の手を 乱数で設定できそうです。

```
// computer の手を 乱数で設定
let computer = Math.floor(Math.random()*3)
```


### 乱数関数を自作する

コンピュータの手を乱数で設定することができました。ここではさらに関数として独立させ分かりやすい名前を付けることにしましょう。

「関数化」することで、一目でプログラムの処理を理解できる、作成したコードの再利用が図れるなどの利点があります。

0, 1, 2 の乱数が得られる「じゃんけんゲーム」専用関数を作っても良いですが、`min(最小値)`と `max(最大値)` を与えることで、双六（すごろく） など他のゲームでも使えるように、**一般化**して作ってみましょう。

**▼幅広い用途に使える一般化された乱数関数**

```
// 乱数関数
// rand(0, 2)と呼ぶと 0, 1, 2 と グーチョキパー の乱数を返す
function rand(min, max){
  return Math.floor(Math.random() * (max - min + 1)) + min
}
```

この乱数関数を使うと、無作為な手を取得するコードは、次のように書くことができます。

```
// computer の手を 乱数で設定
let computer = rand(0, 2)
```

すっきり、分かりやすくなりました。いろいろ関数を自作してみて下さい。


### テンプレートリテラル

乱数の設定が正しくできているかどうか確認するためにも、コンピュータがどの手を出したのか、出力させてみましょう。

```
console.log(computer)
```

と書けば、コンピュータの手を表示させることができます。しかしながら、プレイヤーの手も

```
console.log(player)
```

と書いていました。このままですと、表示された 0, 1, 2 の数字が、コンピューターの手なのかプレイヤーの手なのか、分からなくなりそうです。そこで次のように書いています。

```
// 設定できているか、確認する。
console.log(`computer: ${computer}`)
```

`console.log()`は、引数に指定した値をブラウザのコンソール画面に出力する関数です。そして注目していただきたいのは、``computer: ${computer}`` という記法です。

これはテンプレートリテラルと呼ばれる記法です。テンプレートリテラルは、ダブルクォーテーション(`"`)の代わりに、バッククォート(```)を使用して文字列を作成する方法で、`${変数}` と書くことで、変数の値を埋め込めることが特徴です。

つまり、変数`computer`の値をコンソール画面に「`computer: 変数の値`」という形式で出力するようになります。
