# はじめての JavaScript

:::{.chapter-lead}
HTMLで文書構造を、CSSで装飾を学んできました。この章では、ウェブサイトに「動き」や「双方向性」を加えるプログラミング言語 JavaScript の基礎を学びます。
アルゴリズムやプログラムの考え方から始め、変数・演算子・条件分岐・繰り返しといった基本文法、そしてDOM操作やイベントリスナまで、JavaScriptの全体像を掴んでいきましょう。
:::


## アルゴリズムとプログラム

### アルゴリズムとは

**アルゴリズム**とは、ある問題を解くための手順・方法のことです。料理のレシピや折り紙の折り方のように、「何をどの順番で行えばよいか」を示したものがアルゴリズムです。

古くはユークリッドの互除法（最大公約数を求める手順）や、エラトステネスの篩（素数を見つける手順）など、数学の世界で発展してきました。

### プログラムとは

**プログラム**とは、アルゴリズムをコンピュータが理解できる言語で書き表したものです。コンピュータへの指示書であり、次の3つの要素で成り立ちます。

- **順次処理**: 上から下へ順番に実行する
- **条件分岐**: YESかNOで判断し、処理を分ける
- **繰り返し**: 同じ処理を何度も反復する

この3つを組み合わせることで、あらゆるプログラムを作ることができます。

### プログラミング言語の種類

人間がコンピュータに指示するために作られた言語が**プログラミング言語**です。

- **機械語・アセンブリ言語**: コンピュータが直接理解できる低水準の言語
- **C言語**: 機械に近いところまで書ける汎用言語
- **Ruby**: 楽しくプログラミングできることを重視した言語
- **JavaScript**: ウェブブラウザで動作する言語

JavaScriptは、ウェブサイトに動きを加えるために広く使われています。


## ウェブ制作の三つの言語

ウェブサイトは、3つの言語を組み合わせて作られています。

| 言語 | 役割 |
|------|------|
| **HTML** | 文書構造を記述する（見出し・段落・リストなど） |
| **CSS** | 見た目を装飾する（色・レイアウト・フォントなど） |
| **JavaScript** | 動きや制御を担当する（クリック反応・アニメーションなど） |

HTMLとCSSだけでは静的なページですが、JavaScriptを加えることで、利用者の操作に反応するインタラクティブなサイトが作れます。


## JavaScriptの始め方

### HTMLからの読み込み

JavaScriptは、HTMLから読み込んで使います。

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="utf-8">
    <title>JavaScript練習</title>
    <script src="hello.js" defer></script>
  </head>
  <body>
    <h1>JavaScript練習</h1>
  </body>
</html>
```

`<script src="hello.js" defer></script>` で、`hello.js` を読み込みます。  
`defer` を付けると、HTMLの読み込み完了後にJavaScriptが実行されます。

### 最初のプログラム

```js
// メッセージを表示する
alert("こんにちは。JavaScript")
```

`//` はコメント（説明書き）です。`alert()` は画面にメッセージを表示する関数です。

### コンソール画面

ブラウザの開発者ツール（右クリック →「調査」）を開くと、**コンソール**タブでJavaScriptを直接入力・実行できます。`console.log()` を使うと、コンソールに値を表示できます。

```js
console.log("Hello, World!")
```


## 変数と定数

### 変数（let）

**変数**は、値を入れておく「箱」のようなものです。`let` で宣言します。

```js
let age = 20      // 変数を宣言し、値を代入
age = 21          // 値を更新
```

### 定数（const）

**定数**は、一度決めたら変更できない値です。`const` で宣言します。

```js
const TAX_RATE = 0.10  // 定数は慣習的に大文字で書く
```

### 命名規則

- 変数名には英字・数字・`_`・`$` が使える（数字から始められない）
- 複数語は `playerName` のようにキャメルケースで書く
- 定数は `MAX_VALUE` のように大文字とアンダースコアで書く


## 演算子

### 四則演算と剰余

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `+` | 足し算 | `5 + 3` → `8` |
| `-` | 引き算 | `5 - 3` → `2` |
| `*` | 掛け算 | `5 * 3` → `15` |
| `/` | 割り算 | `5 / 2` → `2.5` |
| `%` | 余り | `5 % 2` → `1` |

### 比較演算子

| 演算子 | 意味 |
|--------|------|
| `===` | 等しい |
| `!==` | 等しくない |
| `<`, `<=` | 小さい、以下 |
| `>`, `>=` | 大きい、以上 |

### 論理演算子

| 演算子 | 意味 |
|--------|------|
| `&&` | かつ（AND） |
| `\|\|` | または（OR） |
| `!` | 否定（NOT） |


## 条件分岐

### if文

条件によって処理を分けるには `if` 文を使います。

```js
let age = 20

if (age >= 18) {
  console.log("大人です")
} else if (age >= 12) {
  console.log("中高生です")
} else {
  console.log("子どもです")
}
```

複数の条件を組み合わせたいときは、論理演算子を用います。次の例では「65歳以上 かつ 会員」のときにシニア割引を適用しています。

```js
let age = 70
let isMember = true

if (age >= 65 && isMember) {
  console.log("シニア会員割引が適用されます")
}
```

### 三項演算子

簡単な条件分岐は、三項演算子で短く書けます。次の例では、18歳以上なら価格を1000円、それ以外なら500円にしています。

```js
let price = (age >= 18) ? 1000 : 500
```


## 繰り返し

### for文

決まった回数の繰り返しには `for` 文を使います。

```js
for (let i = 0; i < 3; i++) {
  console.log(i)  // 0, 1, 2 と表示
}
```

### for...of文

**配列**（いくつかの値をひとまとめにして扱える入れ物）の要素を順に処理するには `for...of` 文が便利です。

```js
let fruits = ["りんご", "みかん", "ぶどう"]
for (const fruit of fruits) {
  console.log(fruit)
}
```

実行すると、コンソールには次のように表示されます。

```text
りんご
みかん
ぶどう
```


## データ型

### 数値型と文字列型

```js
let num = 123          // 数値型
let str = "こんにちは"  // 文字列型
```

### テンプレートリテラル

バッククォート `` ` `` で囲むと、変数を埋め込めます。

```js
let name = "山田"
let age = 20
let message = `名前は${name}です。${age}歳です。`
```


## 配列と連想配列

### 配列

複数の値をまとめて扱うには**配列**を使います。

```js
let scores = [85, 95, 75, 80]
console.log(scores[0])  // 85（0番目の要素）
console.log(scores.length)  // 4（要素数）
```

### 連想配列（オブジェクト）

キーと値のペアで管理するには**連想配列**を使います。

```js
let person = { name: "山田", age: 20 }
console.log(person.name)  // "山田"
console.log(person["age"])  // 20
```


## 関数

### 関数の定義と呼び出し

処理をまとめて名前を付けたものが**関数**です。

```js
// 関数の定義
function add(a, b) {
  return a + b
}

// 関数の呼び出し
let result = add(5, 3)  // 8
```

### アロー関数

ES2015以降、より簡潔に書ける**アロー関数**が使えます。

```js
const add = (a, b) => {
  return a + b
}
```


## オブジェクト指向

### クラスとインスタンス

従来の「手続き型プログラミング」では、幅を表す変数 `width` と高さを表す変数 `height`、面積を求める関数 `area(width, height)` のように、データとそれを処理する関数が別々に置かれがちです。プログラムが大きくなると、「どのデータをどの関数が使っているのか」が分かりにくくなってしまいます。

オブジェクト指向では、関連するデータと処理を 1 つのまとまりとして扱います。これを実現する仕組みが **クラス** と **インスタンス** です。

- **クラス**: どんなデータ（プロパティ）を持ち、どんな操作（メソッド）ができるかを定義した「設計図」
- **インスタンス**: その設計図から実際に作られた「モノ（オブジェクト）」

同じクラスから、幅や高さの違う長方形オブジェクトをいくつでも作ることができ、長方形に関する情報と処理を 1 箇所にまとめて管理できるので、大きなプログラムでも見通しが良くなります。

```js
class Rectangle {
  constructor(width, height) {
    this.width = width
    this.height = height
  }

  area() {
    return this.width * this.height
  }
}

let rect = new Rectangle(10, 20)
console.log(rect.area())  // 200
```

ここでは、長方形の縦横の長さ（幅10・高さ20）を指定して `Rectangle` クラスからインスタンス `rect` を作成し、その `area()` メソッドを呼び出して面積を求めています。

## DOM操作

### DOMとは

**DOM（Document Object Model）**は、JavaScriptからHTMLを操作するための仕組みです。

### 要素の取得

```js
// IDで取得
let elem = document.getElementById("myId")

// CSSセレクタで取得
let elem = document.querySelector(".myClass")
let elems = document.querySelectorAll("li")
```

### 要素の内容を変更

```js
let h1 = document.querySelector("h1")
h1.textContent = "新しいタイトル"  // テキストを変更
h1.innerHTML = "<em>強調</em>"     // HTMLを変更
```

### 要素の作成と追加

```js
let p = document.createElement("p")
p.textContent = "新しい段落"
document.body.append(p)
```


## イベントリスナ

### イベントリスナとは

**イベントリスナ**は、「クリックされたとき」「ページが読み込まれたとき」など、特定の出来事に反応して処理を実行する仕組みです。

```js
// ボタンを取得
let button = document.getElementById("myButton")

// クリック時の処理を登録
button.addEventListener("click", () => {
  alert("ボタンがクリックされました")
})
```

### よく使うイベント

| イベント | タイミング |
|----------|------------|
| `click` | クリック時 |
| `submit` | フォーム送信時 |
| `change` | 値が変化したとき |
| `keydown` | キーが押されたとき |
| `load` | 読み込み完了時 |


## まとめ

この章では、JavaScriptの基礎を学びました。

- **アルゴリズム**は問題を解く手順、**プログラム**はそれをコンピュータ向けに書いたもの
- **変数**（`let`）と**定数**（`const`）でデータを管理する
- **条件分岐**（`if`）と**繰り返し**（`for`）で処理の流れを制御する
- **配列**と**連想配列**で複数のデータをまとめる
- **関数**で処理を再利用可能にする
- **DOM**でHTMLを操作し、**イベントリスナ**でユーザー操作に反応する

詳細な文法や応用的な使い方は、AIに質問したり、[MDN Web Docs](https://developer.mozilla.org/ja/) を参照してください。実際に手を動かしながら、少しずつ習得していきましょう。
