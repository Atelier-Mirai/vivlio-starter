# 終わりに

本書は、計算機の歴史を学び、HTML / CSS / JavaScript の基礎を習得、iPhone でも遊べるじゃんけんゲームを作り上げ、全世界に公開いたしました。

全部で、百行余りのじゃんけんプログラム、要所要所にコメントも付けていますので、今まで学んできた知識で読解できるはずです。ぜひ、遊んでみてください。自分で作ったプログラムの体験はいかがでしょうか。コードを一行一行追いながら、その動作を確認し、理解を深めていく過程は、プログラミング学習における最も重要な体験の一つです。

いろいろ創意工夫して、さまざまなアプリを作っていけそうですね。このじゃんけんゲームを土台として、より複雑なゲームロジックを組み込んだり、美しいアニメーションを追加したり、あるいはまったく異なる種類のアプリケーションに挑戦したりすることもできるでしょう。プログラミングの世界は無限の可能性に満ちています。

ここまでの学習を通じて、皆さんは単なる技術の習得以上のものを得たはずです。問題を分析し、解決策を設計し、それを実装し、検証するという一連の思考プロセスは、プログラミングに限らず、あらゆる創造的活動において役立つ力となります。また、エラーに直面したときに諦めずに原因を探り、解決する粘り強さも身につけられたことでしょう。

「福祉」。「福」「祉[^114]」どちらも「めぐみ、さいわい」という意味を持ちます。

[^114]: 「祉い」と書いて、「さいわい」と読みます。天からの恵みがその身に止まる意味です。

この言葉が示すように、私たちが学ぶ技術は、ただ自分自身のためだけにあるのではありません。それは人々に恵みをもたらし、幸せを届けるための手段なのです。プログラミングという技術も同じです。コードを書く行為は、誰かの課題を解決し、生活を便利にし、笑顔を生み出す可能性を秘めています。

「熱き心、{逞|たくま}しき{腕|かいな}、冷静な頭脳」

学生時代に言われた言葉ですが、福祉を生きる者は、人としての熱い思い、暖かい心を持ち、その上で、冷静な判断力を{以|もっ}て、力強く行動するのだと。

この三つの要素は、優れたエンジニアにも不可欠なものです。熱き心とは、より良いものを作りたいという情熱であり、ユーザーの喜びを自分の喜びとする共感力です。逞しき腕とは、困難な技術的課題に立ち向かう実践力であり、諦めない実行力です。そして冷静な頭脳とは、複雑な問題を論理的に分析し、最適な解決策を見出す思考力です。これら三つが揃って初めて、人の心に届く作品が生まれるのです。

技術の世界は日々進化し続けています。本書で学んだ HTML、CSS、JavaScript は基礎中の基礎ですが、この基礎こそが、どれほど技術が進歩しようとも変わらない本質です。新しいフレームワークやライブラリが次々と登場しても、その根底にある原理は同じです。ですから、ここで築いた土台を大切にしながら、常に学び続ける姿勢を持ち続けてください。

「工学」の「工」は、「天の{理|ことわり}を、地に下ろす」意味です。

技術の産物としての社会ではなく、世界を{耀|かがや}かせるために技術を用いてください。技術に使われるのではなく、技術を使いこなし、人の道に役立てる人となってください。

この言葉は、現代においてますます重要な意味を持ちます。AI や自動化が進む時代だからこそ、技術を「何のために使うのか」という目的意識が問われます。便利さや効率だけを追求するのではなく、その技術が本当に人々の幸せに貢献するのか、社会をより良い方向に導くのかを常に問い続ける必要があります。プログラミングという技術を手にした皆さんには、その力を正しく、そして善く使う責任があるのです。

本書を手に取り、最後まで読み進めてくださったことに、心から感謝申し上げます。途中で難しく感じた部分もあったかもしれません。コードが思い通りに動かず、悩んだ時間もあったでしょう。しかし、その一つ一つの経験が、皆さんを成長させてくれたはずです。

これからも、失敗を恐れずに挑戦し続けてください。エラーメッセージは敵ではなく、上達への道しるべです。動かないコードと格闘する時間は、決して無駄ではありません。その過程で培われる思考力と問題解決能力こそが、皆さんの真の財産となります。

皆さんの手によって、新しい時代にふさわしい、人に寄り添う技術が生まれることを期待しています。そして、その技術が多くの人々に笑顔をもたらし、社会に温もりを届けることを信じています。どうか、学び続け、創り続け、そして人々に恵みを届け続けてください。

令和の御世を生きる皆さんが素晴らしい人生を生き、素晴らしい日本を創ることを願って筆を置きます。

{彌榮|いやさか}
