# 終わりに

本書を最後までお読みくださり、心より感謝申し上げます。

Markdown で原稿を書き、`vs build` コマンドひとつで美しい書籍が生まれる。その一連の流れを、本書を通じて体験していただけたなら幸いです。技術書、解説書、同人誌、社内マニュアル——Vivlio Starter の世界は、あなたが書きたいものの数だけ広がっています。いろいろと創意工夫しながら、ぜひさまざまな書籍づくりに挑戦してみてください。

「福祉」という言葉があります。「福」も「祉[^114]」も、どちらも「めぐみ、さいわい」という意味を持つ美しい言葉です。

[^114]: 「祉い」と書いて「さいわい」と読みます。天からの恵みがその身に止まるという意味です。

わたくし達が身につけた技術を、ただその一身に留めることなく、周りの方々へ伝えてください。原稿を書き、本の形に整え、世に送り出すこと——それは誰かの学びを助け、誰かの視野を広げ、誰かの心に届く営みです。技術を通じて人々に恵みを{齎|もたら}し、幸せを届ける。それもまた、ひとつの福祉のかたちではないでしょうか。

「熱き心、{逞|たくま}しき{腕|かいな}、冷静な頭脳」

学生時代に教わった言葉です。福祉を志す者は、人としての熱い思いと暖かい心を持ち、冷静な判断力を{以|もっ}て力強く行動せよ、と。

執筆活動に{準|なぞら}えるなら——熱き心とは、伝えたいことへの情熱であり、読者の理解を願う誠実さ。{逞|たくま}しき{腕|かいな}とは、原稿と向き合い、書き直し、磨き続ける実践力。そして冷静な頭脳とは、構成を整理し、読者の視点から内容を吟味する思考力。この三つを携えて書かれた言葉は、きっと読み手の心に届くことでしょう。

「工学」の「工」は、天の{理|ことわり}を地に下ろすという意味を持ちます。

技術の産物として社会があるのではなく、世界を{耀|かがや}かせるために技術がある。技術に使われるのではなく、技術を使いこなし、人の道に役立てる——そうありたいと願います。

Vivlio Starter もまた、この精神のもとに作られました。CSS 組版という専門的な技術の恵みを、誰もが手軽に享受できるよう地に下ろすこと。それがこのシステムの目指すところです。

Vivlio Starter は、半年以上の年月をかけて開発してきたシステムです。書き手が「書くこと」だけに向き合えるよう、ひとつひとつ機能を積み重ねてまいりました。至らぬ点もあろうかとは思いますが、皆さまの手で活用され、人に寄り添う素敵な書籍が生まれていくことが、何よりの喜びです。

どうか、書き続けてください。創り続けてください。あなたの言葉が誰かの学びとなり、歩みを支え、恵みを届けるものとなりますように。

令和の御世を生きる皆さんが素晴らしい人生を歩み、素晴らしい日本を創ることを願って、筆を置きます。

{彌榮|いやさか}
