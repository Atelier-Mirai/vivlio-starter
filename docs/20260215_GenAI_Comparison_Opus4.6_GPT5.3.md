## 3. 業務ユースケース別 適合性分析

### ① Ruby Gem / Webサイト開発 (WindSurf利用)
**推奨: Claude Sonnet 4.6 (標準) + GPT-5.3-Codex (UI/UX)**

* **役割分担:**
    * **Claude Sonnet 4.6:** 実力は先代 Opus 以上。1Mトークンを活かし、Ruby on Rails などの大規模なディレクトリ構成を常に「頭に入れた状態」で開発を進められる。**自律的なテスト実行と修正（Computer Use 2.0）** の精度が非常に高い。
    * **GPT-5.3-Codex:** CSSの微調整や、一刻を争うデバッグなど、レスポンスの「リズム」が重要な場面で併用。

### ② 書籍執筆 (本文執筆・表現の磨き上げ)
**推奨: Claude Sonnet 4.6 (下書き・構成) / Claude Opus 4.6 (最終推敲)**

* **理由:**
    * **Sonnet 4.6:** プロットから初稿の書き出しまでを、高速かつ高品質に行う。1M窓により、前後の章との矛盾を自動で検知。
    * **Opus 4.6:** 特に「感動的な表現」や「比喩の独創性」を求める最終推敲段階で投入。文脈の深い読み取り（Adaptive Thinking）により、読者の感情を揺さぶる磨き上げが可能。
