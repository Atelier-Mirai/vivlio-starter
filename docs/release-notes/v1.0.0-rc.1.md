# vivlio-starter v1.0.0-rc.1

**Release Candidate 1**（2026-06-21）

ベータ版から約 50 日にわたる改善を積み重ね、いよいよ 1.0 の正式版に向けたリリース候補版です。
EPUB / Kindle 出力の本格対応、EPUB 生成パイプラインの全面的な品質改善、RC 品質保証テスト群の導入が今回のハイライトです。

> RC 版は正式版の前段です。利用感の最終確認を目的としており、`config/book.yml` の `project.version` など正式版固有の確定作業は次段で行います。

---

## ✨ ハイライト

### 📚 EPUB / Kindle ターゲット分離（目玉機能）
電子書籍出力を配信先ごとに最適化された 2 ターゲットへ分離しました。

- **`epub`（クリーン EPUB）** — Kobo / Apple Books 向け。WebP・高画質 SVG・テーマ色をそのまま維持。
- **`kindle`（KPF）** — Amazon KDP 向け。Kindle（KFX）の CSS 制約に合わせて自動劣化させた中間 EPUB を、`kindlepreviewer` で `.kpf` へ自動変換。
- WebP → JPEG/PNG 変換、数式・扉絵/節絵の SVG / JPEG 出し分け、コードのテーブル化、`::before` ラベルの実体化などを Kindle フレーバ限定で適用。
- **クリーン EPUB・Kindle 中間 EPUB の双方で epubcheck ERROR 0 / WARNING 0 を達成。**

`config/book.yml` の `output.targets` に `kindle` を追加し、`output.kindle.embed`（既定 `false`＝二重表紙回避）で表紙の扱いを制御できます。

### 🔧 EPUB 生成パイプラインの全面改善
- epubcheck の構造 ERROR を **0 件**に（マージンボックス除去・脚注 id 重複除去・table align 変換・NCName 違反 id 補正・用語集 dl 分割 ほか）。
- フォント非埋め込みを既定化し、**EPUB サイズを 322MB → 25MB へ削減**（generic フォールバックで書体カテゴリは維持）。

### 🧮 数式・扉絵の画像化
- LaTeX 数式をビルド時に **静的 SVG へ焼き込み**（MathJax）。PDF・EPUB・表セルを同一 SVG で描画し、Kindle 含む全リーダーで崩れません。
- 扉絵（章扉）・節絵を **合成画像化**。リフロー型 EPUB / Kindle でも飾り絵＋見出しが確実に表示されます。

### 🛡️ 前提条件ガードと doctor の自己修復
- `vs build` 実行前に必須条件を検証し、スタックトレースではなく**行動可能な 🔴 メッセージ**で早期終了。
- `vs doctor --fix` が設定ファイルの欠落・破損を復元（非破壊・`.bak` 退避）、破損 catalog/book.yml を best-effort サルベージ。
- 中断で壊れた Vivliostyle の headless Chrome、ローカル Playwright の Chromium 未取得も検出・修復。

### ✅ RC 品質保証テスト群
- マニュアルフルビルド警告ゼロ / PDF フォント検査（Type 3 排除）/ パッケージング E2E / ファズ / 機能縮退 / NFD / CLI 契約 / ドキュメント整合 / 冪等性 / EPUB(epubcheck) / 依存カナリアの 11 グループを新設。
- 出力ターゲット整合性テスト（`rake test:targets`）と堅牢性テスト 18 項目を追加。

### 📦 配布物の最適化
- gem パッケージをホワイトリスト化し **430MB → 55MB**。未参照フォント（Noto CJK OTF・BIZ UD 系）を削除。

## 🐞 主なバグ修正（抜粋）
- `print_pdf` 単独ターゲットで本文が欠落する不具合を修正。
- 脚注 URL の重複表示（sideimage 内・テーブル内リンク）を修正。
- 画像配置（`align=left/center/right`）の連鎖崩れを修正。
- 付録のラベル（A/B/C…）・図表番号（D-1 等）が catalog 順と一致するよう修正。
- PDF アウトラインのしおりが目次ページへ飛ぶ不具合を修正。
- `vivlio-starter-pdf` プラグインが `bundle exec` 配下で自動検出されない不具合を修正。

## 🔒 セキュリティ / 堅牢性
- `catalog.yml` / `data/*.yml` の `YAML.safe_load` 化（危険タグ拒否）。
- 原稿内の危険スキーム（`file://` / `javascript:`）検出（常時有効）。
- 画像パスのディレクトリトラバーサル・HTML 特殊文字対策。
- SIGINT / SIGTERM の graceful handling（終了コード 130 / 143）。

## ⚠️ 既知の不具合
詳細は [`docs/specs/KNOWN_ISSUES.md`](../specs/KNOWN_ISSUES.md) を参照してください。

- EPUB 本文の脚注記号「†」数が、同時にビルドする target に依存する（本文の実体テキストは同一）。
- EPUB(Kindle) の表内インライン数式のサイズが不安定（表内に数式を置かない運用を推奨。ディスプレイ `$$` は正常）。
- EPUB(Kindle) のコード行番号と行の対応がずれる場合がある（将来対応）。

## 📥 インストール / 動作要件
- Ruby **4.0 以上**（正式版で要件緩和を検討中）。
- Kindle（KPF）出力には **Kindle Previewer 3**（`kindlepreviewer` CLI）が必要です。未導入時は中間 EPUB を残し、KPF 変換のみスキップします。

```bash
gem install vivlio-starter --pre   # rubygems.org 公開時
# もしくは GitHub から直接インストール
```
